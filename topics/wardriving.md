# Wardriving (Вардинг)

## Описание
Wardriving — практика сканирования беспроводных сетей Wi-Fi для обнаружения доступных сетей и определения их параметров. Может применяться как для улучшения безопасности, так и для кибератак.

## Уровень
Средний

## Примеры использования
- Сканирование беспроводных сетей для обнаружения доступных точек доступа.
- Определение параметров сетей (SSID, тип шифрования, мощность сигнала).
- Сбор данных для анализа безопасности Wi-Fi сетей.
- Использование для улучшения безопасности собственных сетей.

## Ресурсы для изучения
- Инструменты для wardriving
- Методики сканирования беспроводных сетей
- Best practices для защиты Wi-Fi сетей
