# Доступ через FTP

## Описание
Использование FTP (File Transfer Protocol) для доступа к файлам на сервере, загрузки эксплойтов и получения данных авторизации из конфигурационных файлов.

## Уровень
Средний

## Примеры использования
- Подключение к FTP серверу с найденными учетными данными.
- Загрузка эксплойтов и инструментов на целевую систему.
- Скачивание конфигурационных файлов для анализа.
- Передача файлов между атакующей и целевой машинами.

## Ресурсы для изучения
- FTP клиенты и команды
- Методики эксплуатации FTP серверов
- Best practices для защиты FTP серверов
