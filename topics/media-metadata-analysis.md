# Работа с метаданными медиафайлов

## Описание
Работа с медиафайлами в OSINT: поиск первоисточника, отслеживание цепочек и хронологии распространения, извлечение и изучение метаданных (EXIF), верификация медиафайлов для определения подлинности.

## Уровень
Средний

## Примеры использования
- Извлечение EXIF данных из фотографий.
- Поиск первоисточника медиафайлов.
- Отслеживание цепочек распространения контента.
- Верификация медиафайлов для определения фейков.
- Анализ метаданных для получения информации о месте и времени создания.

## Ресурсы для изучения
- Инструменты для извлечения EXIF данных
- Методики верификации медиафайлов
- Best practices для работы с метаданными
