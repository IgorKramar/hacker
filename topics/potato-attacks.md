# Картофельные атаки (Potato Attacks)

## Описание
Картофельные атаки — разновидность атаки с использованием учетных данных для эскалации привилегий в Windows системах. Используют учетные данные для аутентификации через NTLM на конечной точке TCP, которую злоумышленник контролирует, что позволяет получить доступ к привилегированным учетным записям.

## Уровень
Продвинутый

## Примеры использования

### Принцип работы
- Картофельная атака использует учетные данные для аутентификации через NTLM на конечной точке TCP, которую злоумышленник контролирует.
- Простой способ вызвать эскалацию в системе для получения доступа к привилегированным учетным записям.
- Использование различных привилегий в этих атаках и их влияние на систему.

### Использование Metasploit
- Metasploit — распространенный и простой способ проведения картофельной атаки.
- Может быть использован для получения доступа к привилегированным учетным записям.
- Автоматизация процесса атаки через Metasploit модули.

### Процесс атаки
1. Получение начальной точки опоры на системе.
2. Настройка контролируемой конечной точки TCP.
3. Принуждение системы к аутентификации через NTLM на контролируемую конечную точку.
4. Перехват учетных данных и использование их для эскалации привилегий.

## Типы картофельных атак

### Hot Potato
- Оригинальная картофельная атака.
- Использование NTLM relay для эскалации привилегий.

### Rotten Potato
- Улучшенная версия картофельной атаки.
- Использование дополнительных техник для повышения эффективности.

### Juicy Potato
- Ещё одна разновидность картофельной атаки.
- Использование различных привилегий и техник.

## Защита от картофельных атак
- Ограничение привилегий пользователей и процессов.
- Мониторинг подозрительной сетевой активности.
- Отключение ненужных привилегий.
- Регулярное обновление системы и применение патчей.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных системах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
