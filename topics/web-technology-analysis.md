# Веб-анализ технологий

## Описание
Определение используемых технологий на веб-сайте: CMS, фреймворки, серверное ПО, библиотеки для выявления потенциальных уязвимостей и векторов атаки.

## Уровень
Средний

## Примеры использования
- Определение CMS (WordPress, Joomla, Drupal).
- Выявление используемых фреймворков и библиотек.
- Определение версий серверного ПО.
- Подмена User-Agent для обхода блокировок.

## Ресурсы для изучения
- Wappalyzer, WhatWeb инструменты
- Методики веб-анализа
- User-Agent подмена техники
