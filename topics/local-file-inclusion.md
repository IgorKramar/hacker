# Local File Inclusion (LFI)

## Описание
Уязвимость Local File Inclusion позволяет читать локальные файлы на сервере через веб-приложение, часто приводя к раскрытию конфиденциальной информации.

## Уровень
Основы

## Примеры использования
- Эксплуатация LFI через параметры URL.
- Чтение конфигурационных файлов (/etc/passwd).
- Обход фильтров и использование path traversal.
- Манипулирование переменными, ссылающимися на файлы, для получения доступа к файлам за пределами корневой папки веб-сайта.
- Пример: загрузка файла с изменённым именем для доступа к файлам паролей.
- Необходимость перебора всех возможных комбинаций для достижения цели.
