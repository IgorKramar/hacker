# Enumeration с помощью скриптов (LinPEAS)

## Описание
Использование автоматизированных скриптов enumeration, таких как LinPEAS, для поиска векторов повышения привилегий.

## Уровень
Средний

## Примеры использования
- Запуск LinPEAS на целевой системе.
- Анализ результатов enumeration.
- Использование найденных векторов для privilege escalation.

## Ресурсы для изучения
- [LinPEAS GitHub](https://github.com/carlospolop/PEASS-ng)
- Методики автоматизированного enumeration
