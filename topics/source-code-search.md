# Поиск исходного кода в репозиториях

## Описание
Поиск исходного кода по ключевым словам в публичных репозиториях (GitHub, GitLab) для обнаружения уязвимых участков кода, секретов и конфиденциальной информации.

## Уровень
Средний

## Примеры использования
- Поиск уязвимых участков кода в библиотеках и репозиториях.
- Обнаружение секретов и API-ключей в публичных репозиториях.
- Анализ кода на наличие уязвимостей.
- Поиск должен быть согласован с владельцами репозитория.

## Ресурсы для изучения
- CodeCrawler и аналогичные инструменты
- GitHub Advanced Search
- Методики безопасного поиска в репозиториях
