# Tor сеть

## Описание
Tor (The Onion Router) — распределенная сеть для анонимной коммуникации, маршрутизация трафика через несколько узлов для сокрытия источника и назначения.

## Уровень
Средний

## Примеры использования
- Анонимный просмотр веб-сайтов через Tor Browser.
- Настройка SOCKS5 прокси через Tor (порт 9050).
- Автоматическая смена IP через Tor сеть.
- Обход блокировок и цензуры.

## Ресурсы для изучения
- [Tor Project](https://www.torproject.org/)
- [Tor Documentation](https://support.torproject.org/)
- Руководства по настройке Tor
