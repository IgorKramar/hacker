# Проброс портов (port forwarding/tunneling)

## Описание
Техники проброса портов и туннелирования для доступа к внутренним сервисам через промежуточные хосты.

## Уровень
Продвинутый

## Примеры использования
- Локальный проброс портов (SSH -L).
- Удаленный проброс портов (SSH -R).
- Использование инструментов туннелирования (Chisel).

- Документация по SSH туннелированию
- Инструменты для port forwarding
