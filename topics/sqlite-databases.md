# Работа с базами данных (SQLite)

## Описание
SQLite — легковесная реляционная база данных, часто используемая в веб-приложениях. Работа с .db файлами, извлечение данных и хэшей.

## Уровень
Средний

## Примеры использования
- Открытие и анализ .db файлов.
- Извлечение хэшей паролей из базы данных.
- SQL-запросы для получения пользовательских данных.

## Ресурсы для изучения
- [SQLite Documentation](https://www.sqlite.org/docs.html)
- SQLite Browser инструменты
