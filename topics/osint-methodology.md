# Методология OSINT

## Описание
Методология проведения OSINT-расследований: декомпозиция задач (метод 5W1H), принципы получения доказательств, системный подход к сбору и анализу информации из открытых источников.

## Уровень
Средний

## Примеры использования
- Использование метода 5W1H для декомпозиции задач расследования.
- Применение принципов получения доказательств (подтверждение, опровержение, фиксация).
- Систематизация процесса сбора информации.
- Контроль процесса и оценка результатов расследования.

## Ресурсы для изучения
- Методология декомпозиции задач
- Принципы работы с доказательствами
- Учебники по уголовному процессу для понимания доказательств
