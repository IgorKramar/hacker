# Анализ SSL/TLS сертификатов

## Описание
Анализ SSL и TLS сертификатов для выявления потенциально опасных сертификатов, аудита безопасности и расследования фишинговых атак.

## Уровень
Средний

## Примеры использования
- Поиск сертификатов, связанных с доменами.
- Выявление подозрительных или скомпрометированных сертификатов.
- Аудит безопасности SSL/TLS конфигураций.
- Расследование фишинговых атак через анализ сертификатов.

## Ресурсы для изучения
- Certy и аналогичные инструменты
- Методики анализа SSL/TLS сертификатов
- Best practices для управления сертификатами
