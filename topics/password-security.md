# Защита паролей

## Описание
Методы защиты паролей: использование соли (salt) для хэширования, предотвращение расшифровки украденных хэшей, рекомендации по использованию сложных паролей и двухфакторной аутентификации.

## Уровень
Средний

## Примеры использования
- Использование соли для защиты паролей при хэшировании.
- Предотвращение использования украденных хэшей паролей.
- Рекомендации по созданию сложных паролей.
- Использование двухфакторной аутентификации (2FA).

## Ресурсы для изучения
- Методики безопасного хэширования паролей
- Best practices для хранения паролей
- Двухфакторная аутентификация
