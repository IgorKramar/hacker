# Атаки на локальные сети

## Описание
Атаки на локальные сети — техники взлома устройств в локальной сети через ARP spoofing, DNS spoofing и перехват трафика. Позволяют злоумышленникам прослушивать действия пользователей, перенаправлять запросы на вредоносные сайты и получать доступ к конфиденциальной информации.

## Уровень
Продвинутый

## Типы атак

### Обнаружение устройств
- Сканирование локальной сети для выявления активных хостов.
- Перечисление всех устройств с указанием MAC-адресов и IP-адресов.
- Определение типов устройств и их операционных систем.
- Подготовка к атаке с подменой ARP.

### ARP Spoofing (подмена ARP)
- Использование IP-адреса устройства для подмены ARP.
- Запуск атаки с подменой ARP для перенаправления всех соединений на маршрутизатор.
- Возможность видеть все действия на целевых устройствах.
- Перехват трафика между устройствами и маршрутизатором.

### DNS Spoofing (подмена DNS)
- Перенаправление пользователей на вредоносные веб-сайты.
- Подмена DNS-запросов для контроля трафика.
- Использование веб-серверов (Apache) для создания вредоносных сайтов.
- Перенаправление на фишинговые страницы.

### Перехват трафика
- Мониторинг сетевого трафика в реальном времени.
- Перехват HTTP/HTTPS запросов.
- Анализ данных, передаваемых по сети.
- Захват учетных данных и конфиденциальной информации.

## Уязвимые сети

### Общедоступные Wi-Fi сети
- Отели и библиотеки уязвимы для атак.
- Отсутствие шифрования трафика в открытых сетях.
- Легкий доступ злоумышленников к сети.

### Домашние сети
- Неправильная настройка безопасности.
- Слабые пароли Wi-Fi.
- Разрешение незнакомым людям подключаться к сети.

## Инструменты для атак

### Bettercap
- Мощный инструмент для проведения атак на сети Wi-Fi и Bluetooth.
- Поддержка ARP spoofing и DNS spoofing.
- Обнаружение устройств в сети.
- Перехват и анализ трафика.

## Защита от атак

### Для пользователей
- Никогда не позволяйте никому подключаться к вашей сети Wi-Fi без доверия.
- Установите правильный пароль для защиты сети.
- Использование VPN для шифрования трафика.
- VPN защищает данные и предотвращает перехват трафика.
- Важность использования VPN для безопасности в любой сети.

### Для администраторов сетей
- Мониторинг ARP-таблиц на предмет подозрительной активности.
- Использование статических ARP-записей.
- Настройка систем обнаружения вторжений (IDS).
- Регулярная проверка сетевой активности.
- Ограничение доступа к сети.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных сетях или с явного разрешения.
- Понимание юридических последствий незаконного использования.
- Атаки на чужие сети являются незаконными.
