# Фишинг с QR-кодами

## Описание
Фишинг с QR-кодами — эволюция социальной инженерии, использующая доверие к безобидным изображениям. Злоумышленники создают QR-коды для перенаправления на поддельные сайты с целью кражи учётных данных, доступа к аккаунтам в социальных сетях и личной информации.

## Уровень
Продвинутый

## Особенности фишинга с QR-кодами
- QR-коды незаметны и быстро сканируются.
- Невозможно отличить вредоносный QR-код от безопасного визуально.
- Использует доверие пользователей к безобидным изображениям.
- Быстрое распространение через печатные материалы и цифровые носители.

## Примеры использования

### Процесс атаки
- Создание фишингового сайта, имитирующего настоящий (например, Instagram).
- Генерация QR-кода, ведущего на поддельный сайт.
- Размещение QR-кода в общественных местах или через цифровые каналы.
- Жертва сканирует QR-код и вводит учётные данные на поддельном сайте.
- Злоумышленник получает учётные данные в реальном времени.

### Технические детали
- Использование инструментов для создания фишинговых сайтов (Medusa Fisher).
- Настройка туннелирования для размещения фишинговой страницы через интернет.
- Генерация QR-кодов с вредоносными ссылками.
- Сохранение учётных данных жертвы.

## Защита от фишинга с QR-кодами

### Для пользователей
- Будьте осторожны при сканировании QR-кодов из неизвестных источников.
- Внимательно изучайте URL после сканирования QR-кода.
- Избегайте ввода конфиденциальной информации на подозрительных сайтах.
- Проверяйте подлинность сайта перед вводом данных.
- Будьте особенно осторожны с QR-кодами, которые требуют немедленных действий или заявляют о срочных ситуациях.

### Для компаний и организаций
- Внедрение систем проверки QR-кодов.
- Обучение сотрудников о рисках безопасности QR-кодов.
- Использование фирменных QR-кодов, которые сложнее подделать.
- Обновление устройств до последней версии (некоторые атаки используют уязвимости в мобильных ОС).
- Регулярное обучение персонала по вопросам кибербезопасности.

## Важные предупреждения
- Использование методов фишинга против реальных людей незаконно.
- Видео и материалы предназначены только для образовательных целей.
- Тестирование только на собственных системах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
