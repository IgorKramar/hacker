# Поиск и анализ документов (Document OSINT)

## Описание
Поиск и анализ документов в OSINT: поиск по ключевым словам и наличию документа, работа с различными форматами файлов, извлечение метаданных, анализ содержимого документов для получения конкретной информации (фамилии, адреса, названия организаций, финансовая информация).

## Уровень
Средний

## Виды поиска документов
- **Поиск по ключевым словам**: знание содержимого документа (списки учеников, участников олимпиад).
- **Поиск по наличию документа**: неопределённое содержимое, изучение сайта на предмет информации о владельце.

## Форматы документов
- DOC, DOCX (Microsoft Word)
- XLS, XLSX (Microsoft Excel)
- PPT, PPTX (Microsoft PowerPoint)
- ODT, ODP (OpenDocument)
- PDF (Portable Document Format)

## Примеры использования

### Поиск документов с Google Dorks
- Основной оператор: `filetype:docx` для поиска документов Word.
- Комбинирование с другими операторами: `filetype:pdf site:example.com`.
- Поиск по содержимому: `filetype:pdf "ключевое слово"`.
- Уточнение запросов для уменьшения количества результатов.

### Утилиты для поиска документов
- **Meta-GoFile**: выкачка файлов и создание отчёта на основе метаданных.
- **DorkDump**: проверка небольших сайтов, отображение метаданных в терминале.

### Специализированные поисковые сайты
- **Intellex**: упрощение написания Google Dorks.
- **FileFish**: применение Google Dorks к введённому сайту.
- **PDF-Fill**: поиск PDF-файлов с возможностью открытия в онлайн-редакторе.
- **DeDigger**: поиск общедоступных файлов на Google Дисках.
- **SearchFTP**: поиск файлов на FTP-серверах.

### Работа с найденными документами

#### Google Pinpoint
- Создание рабочего пространства и добавление документов.
- Поддержка различных форматов: PDF, изображения, офисные форматы, веб-страницы, текстовые документы, аудио и видеофайлы.
- Разделение больших файлов на части.
- Распознавание текста на изображениях и в документах (OCR).
- Преобразование аудиофайлов в текст.
- Создание текстовой версии видео.
- Поиск по всем загруженным файлам.
- Расшифровка аббревиатур и поиск синонимов.
- Выделение, копирование и создание ссылок на найденный текст.

#### Voyant Tools
- Словарный анализ текста.
- Загрузка документа или добавление ссылки.
- Показ частоты использования слов и фраз.
- Анализ упоминаний слов в тексте.

#### Soda PDF
- Онлайн-редактор PDF с конвертером в разные форматы.
- Поиск по документу.
- Сравнение двух документов.
- Отметки на тексте и выделение фрагментов.

### Анализ документов
- Фиксация неизменяемых данных: анкетные данные, названия, даты, информация о событиях.
- Извлечение метаданных из документов.
- Работа с большими документами требует специальных инструментов (Control F не всегда эффективен).

## Особенности
- Документы могут храниться в виде изображений, что затрудняет поиск.
- Иногда документы размещают в виде изображений специально для усложнения поиска.
- Важно правильно изучать документы и выбирать нужную информацию.

## Ресурсы для изучения
- Google Dorks для поиска документов
- Google Pinpoint для работы с объёмными документами
- Voyant Tools для словарного анализа
- Soda PDF для редактирования PDF
- Методики извлечения метаданных из документов
