# Фазинг (fuzzing) эндпоинтов

## Описание
Автоматизированный поиск скрытых эндпоинтов, параметров и уязвимостей через массовую отправку запросов с различными входными данными.

## Уровень
Средний

## Примеры использования
- Поиск скрытых директорий и файлов.
- Фазинг параметров API.
- Обнаружение уязвимостей через автоматизированные запросы.

## Ресурсы для изучения
- Инструменты: Gobuster, Dirbuster, ffuf
- Методики фазинга веб-приложений
