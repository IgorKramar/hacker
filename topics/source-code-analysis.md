# Просмотр исходного кода страниц

## Описание
Анализ исходного кода веб-страниц для поиска уязвимостей, скрытых комментариев, API-ключей и другой чувствительной информации.

## Уровень
Основы

## Примеры использования
- Просмотр HTML-кода страницы (Ctrl+U).
- Анализ JavaScript-файлов.
- Поиск комментариев с чувствительными данными.

## Ресурсы для изучения
- OWASP Web Security Testing Guide
- Инструменты для анализа кода
