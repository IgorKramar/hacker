# DNS анализ

## Описание
Анализ DNS (Domain Name System) для получения информации о домене: IP-адреса, поддомены, MX-записи, TXT-записи, серверы имен и дата регистрации.

## Уровень
Средний

## Примеры использования
- Получение информации о владельце домена через whois.
- Изучение DNS записей через dig.
- Получение IP-адресов и поддоменов.
- Анализ MX-записей для email серверов.
- Изучение TXT-записей для получения дополнительной информации.

## Ресурсы для изучения
- whois и dig команды
- Методики DNS анализа
- DNS enumeration техники
