# Использование нейросетей и ИИ в OSINT

## Описание
Использование нейронных сетей и искусственного интеллекта для улучшения поиска и анализа информации в OSINT: специализированные поисковики на основе нейросетей, обучение моделей для конкретных задач (геолокация по фотографиям).

## Уровень
Средний

## Примеры использования

### Поисковые системы на основе нейросетей
- **Perplexia**: сервис на основе нейросетей для удобного поиска.
- **Яндекс Нейропоиск**: улучшенный поиск с использованием ИИ.
- Нейросети можно натаскивать на определённые задачи.

### Специализированные задачи
- **GPS-агент**: нейросеть, натренированная на игре GeoGuessr, точно угадывает местность по характерным признакам на фотографиях.
- Обучение моделей для угадывания местности по фотографиям.
- Использование OpenAI API для тестирования GPS-агента.

### Преимущества использования ИИ
- Удобство поиска информации.
- Возможность обучения на конкретных задачах.
- Точность определения местоположения по визуальным признакам.

## Ресурсы для изучения
- Perplexia и аналогичные сервисы
- Яндекс Нейропоиск
- OpenAI API для OSINT задач
- Методики обучения нейросетей для OSINT
