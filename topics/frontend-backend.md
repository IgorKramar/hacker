# Фронтенд и бэкенд

## Описание
Разделение веб-разработки на фронтенд (клиентская часть) и бэкенд (серверная часть), их роли и взаимодействие.

## Уровень
Основы

## Примеры использования
- Разработка пользовательского интерфейса (фронтенд).
- Создание серверной логики (бэкенд).
- API для взаимодействия между слоями.
