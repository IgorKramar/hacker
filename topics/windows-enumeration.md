# Перечисление систем Windows (Windows Enumeration)

## Описание
Перечисление систем Windows — ключевой аспект повышения привилегий, включающий сбор информации о системе, версии ОС, архитектуре, пользователях, сетях, сервисах и конфигурации для поиска уязвимостей.

## Уровень
Средний

## Ручное перечисление

### Системная информация
- Использование системной информации для получения информации о системе.
- Получение имени хоста, операционной системы, версии ОС, архитектуры и других данных.
- Использование команды `wmic` для получения информации о дисках и исправлениях системы.

### Перечисление пользователей
- Различные типы пользователей и их привилегии.
- Перечисление пользователей на компьютере.
- Использование команды `whoami /priv` для просмотра доступных привилегий.

### Перечисление сетей
- Использование команд `ipconfig` и `arp -a` для получения информации о сети и таблице маршрутизации.
- Важность ознакомления с таблицей ARP и таблицей маршрутизации для понимания сетевой архитектуры.
- Анализ сетевых подключений и открытых портов.

### Поиск паролей
- Хранение паролей в файлах SAM, реестре, VNC, INI, реестре и полезной нагрузке.
- Поиск паролей в конфигурационных файлах и других ресурсах.
- Анализ реестра для поиска сохранённых паролей.

### Обнаружение брандмауэров и антивирусов
- Управление сервисами с помощью команды `sc`.
- Запрос информации о защитнике Windows и других службах.
- Использование команды `netsh advfirewall` для получения информации о брандмауэре и его состоянии.

## Автоматизированные инструменты

### WinPEAS
- Скрипт для повышения привилегий в Windows с цветовой координацией.
- Выполняет автоматическое перечисление и выявляет уязвимости.
- Анализ процессов, сервисов, сетевой информации.

### Watson
- Инструмент для проверки уязвимостей в Windows.
- Поиск известных уязвимостей в системе.

### Sherlock
- Инструмент для поиска уязвимостей и перечисления.
- Проверка системы на наличие известных уязвимостей.

### PowerUp
- PowerShell скрипт для поиска уязвимостей.
- Автоматизированное перечисление и поиск векторов эскалации.

### Jaws
- Инструмент для проверки уязвимостей.
- Анализ системы на предмет слабых мест.

### Windows Exploit Suggester
- Программа для поиска эксплойтов для Windows (написана на Python).
- Сопоставление версии системы с известными уязвимостями.

## Важность перечисления
- Перечисление — ключевой аспект повышения привилегий.
- Необходимость пошагового подхода и использования нескольких инструментов.
- Важность знания всей информации о системе перед началом эксплуатации.
