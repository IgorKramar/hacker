# Перехват и расшифровка Wi-Fi handshake

## Описание
Перехват и расшифровка handshake (рукопожатия) между устройством и роутером для получения доступа к Wi-Fi сети: перехват сообщений SYN и ACK, деаутентификация устройств, расшифровка с помощью словарей паролей.

## Уровень
Продвинутый

## Примеры использования

### Процесс handshake
- Handshake — рукопожатие между устройством и роутером.
- Включает отправку и получение сообщений SYN и ACK.
- Перехват handshake позволяет злоумышленнику получить доступ к сети.

### Практическое выполнение
- Настройка сетевых адаптеров в режиме монитора.
- Запуск сетевого адаптера в режиме монитора для мониторинга сетей.
- Сканирование сетей и сбор данных для перехвата handshake.
- Деаутентификация устройств для принудительного переподключения.
- Перехват handshake при переподключении устройства.

### Расшифровка handshake
- Расшифровка handshake с помощью словаря паролей.
- Использование инструментов для брутфорса паролей Wi-Fi.
- Анализ перехваченных данных для извлечения ключа.
- Использование списков слов (rockyou.txt) для угадывания паролей.
- Более целенаправленные списки слов для профессиональных хакеров.
- Программное обеспечение перебирает множество паролей, пока не найдет правильный.
- Наличие четырехстороннего handshake не гарантирует получение пароля.
