# Настройка виртуальных машин для пентестинга

## Описание
Настройка виртуальных машин (Kali Linux, Metasploit Machine) для безопасной практики пентестинга в изолированной среде, создание лабораторной среды для обучения.

## Уровень
Основы

## Примеры использования
- Создание виртуальных машин в VirtualBox для пентестинга.
- Настройка Kali Linux и целевых машин (Metasploit Machine).
- Практика на виртуальных машинах без риска для реальных систем.
- Создание изолированной лабораторной среды для обучения.

## Ресурсы для изучения
- VirtualBox документация
- Kali Linux установка и настройка
- Методики создания лабораторных сред для пентестинга
