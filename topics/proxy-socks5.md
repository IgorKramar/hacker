# Настройка прокси (SOCKS5)

## Описание
Настройка прокси-серверов, в частности SOCKS5, для маршрутизации трафика через промежуточные серверы, обеспечение анонимности и обход ограничений.

## Уровень
Средний

## Примеры использования
- Настройка SOCKS5 прокси в браузере.
- Использование Tor как SOCKS5 прокси (localhost:9050).
- Настройка прокси для приложений и скриптов.
- Предотвращение утечки DNS через SOCKS5.

## Ресурсы для изучения
- Документация по SOCKS5 протоколу
- Руководства по настройке прокси в браузерах
- Методики предотвращения DNS утечек
