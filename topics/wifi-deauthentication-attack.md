# Атака деаутентификации Wi-Fi

## Описание
Атака деаутентификации для принудительного отключения устройств от Wi-Fi сети с целью перехвата handshake при переподключении или создания помех в работе сети.

## Уровень
Средний

## Примеры использования

### Процесс атаки
- Использование команды для отправки сообщений об отключении доступа.
- Маршрутизатор может отправлять сообщения об отключении доступа, которые хакер может использовать.
- Настройка таргетинга на одного пользователя или всю сеть.
- Принудительное отключение устройств от сети.

### Инструменты
- Использование команд для деаутентификации (airplay-ng).
- Flipper Zero для легкого сканирования сети и запуска атаки.
- Профессиональные инструменты для более сложных атак.

### Цели атаки
- Принуждение устройств к переподключению для перехвата handshake.
- Создание помех в работе сети.
- Подготовка к другим атакам (перехват handshake, Evil Twin).

## Защита от деаутентификации
- Использование защищенных протоколов (WPA3).
- Мониторинг сетевой активности на предмет подозрительных действий.
- Использование корпоративных сетей с защитой от деаутентификации.
