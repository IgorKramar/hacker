# Поиск уязвимых устройств в интернете

## Описание
Техники поиска и анализа подключенных к интернету устройств (IoT, камеры, серверы) с использованием специализированных поисковых систем для обнаружения уязвимостей и незащищенных сервисов.

## Уровень
Средний

## Примеры использования
- Поиск незащищенных камер наблюдения и IoT устройств.
- Обнаружение серверов с открытыми портами и уязвимостями.
- Мониторинг сетевых активов организации.
- Исследование распространения уязвимых устройств в интернете.

## Ресурсы для изучения
- Shodan и аналогичные поисковые системы
- Методики поиска устройств по метаданным
- Best practices для защиты устройств от обнаружения
