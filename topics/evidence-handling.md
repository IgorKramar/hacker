# Работа с доказательствами в OSINT

## Описание
Система работы с доказательствами в OSINT-расследованиях: фиксация доказательств (скриншоты, ссылки на источники), хранение информации, создание чек-листов, запись процесса для повторения.

## Уровень
Средний

## Примеры использования
- Сохранение данных в заметке с ссылкой на источник.
- Создание скриншотов страниц с видимым адресом.
- Запись способа нахождения информации.
- Использование записи экрана или софта для логирования действий.
- Создание единого подхода к хранению информации для каждого расследования.

## Ресурсы для изучения
- Методики фиксации доказательств
- Инструменты для записи процесса
- Стандарты работы с доказательствами (протокол Беркли, IC-206-01)
