# REST API безопасность

## Описание
Безопасность REST API: следование правилам REST (использование POST, PUT, PATCH, DELETE для изменяющих операций), защита от CSRF, безопасное взаимодействие фронтенда с бэкендом.

## Уровень
Средний

## Примеры использования
- Использование правильных HTTP методов для операций.
- Защита от CSRF через токены и кукисы.
- Обеспечение безопасного взаимодействия фронтенда с бэкендом.
- Следование принципам REST для снижения уязвимости к атакам.

## Ресурсы для изучения
- REST API best practices
- Методики защиты API от CSRF
- Безопасная архитектура веб-приложений
