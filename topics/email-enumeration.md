# Поиск email-адресов (Email Enumeration)

## Описание
Техники поиска email-адресов, связанных с доменами и организациями, для использования в фишинговых кампаниях, маркетинге, рекрутинге и OSINT исследованиях.

## Уровень
Средний

## Примеры использования
- Поиск email-адресов сотрудников организации по домену.
- Верификация подлинности email-адресов.
- Сбор контактной информации для OSINT.
- Использование в этичных пентестах для социальной инженерии.

## Ресурсы для изучения
- Hunter и аналогичные инструменты
- Методики email enumeration
- Best practices для защиты от email enumeration
