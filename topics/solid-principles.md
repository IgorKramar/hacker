# Принципы проектирования SOLID

## Описание
Принципы SOLID для проектирования кода: Single Responsibility, Open/Closed, Liskov Substitution, Interface Segregation, Dependency Inversion.

## Уровень
Средний

## Примеры использования
- Применение принципов при проектировании классов.
- Рефакторинг кода с учетом SOLID.

## Ресурсы для изучения
- Книга: "Clean Code" by Robert C. Martin
- Статьи по SOLID принципам
