# Атака Man-in-the-Middle (MITM)

## Описание
Атака "человек в центре" (Man-in-the-Middle) — простая, но эффективная атака, включающая ARP-спуфинг для перехвата и контроля сетевого трафика между жертвой и роутером.

## Уровень
Средний

## Примеры использования

### ARP-спуфинг
- Обман Wi-Fi роутера, выдача себя за цель.
- ARP-спуфинг позволяет хакерам контролировать трафик.
- Использование Wireshark для перехвата данных.
- Перенаправление трафика через атакующую машину.

### Процесс атаки
- Использование ноутбука и Kali Linux для взлома.
- Настройка сетевого адаптера в режиме мониторинга.
- Выполнение ARP-спуфинга для перехвата трафика.
- Анализ перехваченных данных с помощью Wireshark.

## Инструменты для MITM атак
- **Bettercap**: мощный инструмент для проведения ARP spoofing и DNS spoofing.
- Специализированные инструменты для выполнения ARP-спуфинга.
- Настройка сетевых параметров для перехвата трафика.
- Использование инструментов для анализа перехваченного трафика.

## Защита от MITM
- Использование VPN для шифрования трафика.
- Корпоративные сети могут изолировать хосты и предотвращать MITM атаки.
- Проверка сертификатов при подключении к защищенным сайтам.

## Ресурсы для изучения
- Методики ARP-спуфинга
- Wireshark для анализа трафика
- Best practices для защиты от MITM атак
