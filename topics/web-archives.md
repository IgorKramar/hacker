# Веб-архивы (Wayback Machine)

## Описание
Использование веб-архивов для поиска старых версий сайтов, обнаружения удаленного контента, административных панелей и чувствительной информации, которая была доступна ранее.

## Уровень
Средний

## Примеры использования
- Поиск старых версий сайта в Wayback Machine.
- Фильтрация страниц, связанных с админкой.
- Обнаружение удаленного контента и файлов.
- Получение исторических данных о сайте.

## Ресурсы для изучения
- [Wayback Machine](https://web.archive.org/)
- Инструменты для работы с веб-архивами
- Методики поиска в архивах
