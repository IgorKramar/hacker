# Языки программирования

## Описание
Компилируемые и интерпретируемые языки программирования, машинный код и процесс трансляции кода.

## Уровень
Основы

## Примеры использования
- Выбор языка для конкретной задачи.
- Понимание различий между компиляцией и интерпретацией.

## Ресурсы для изучения

- Книга: "Programming Language Pragmatics"
