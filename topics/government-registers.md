# Государственные реестры и базы данных

## Описание
Использование государственных реестров и открытых баз данных для сбора информации: данные о юридических и физических лицах, судебные решения, миграционные документы, розыск, наследственные дела.

## Уровень
Средний

## Примеры использования

### Данные о юридических лицах
- ФНС (Федеральная налоговая служба) — информация о компаниях.
- Судебные приставы — информация о долгах.
- Реестры патентов и товарных знаков.
- Удостоверения частных охранников.
- Зарегистрированные СМИ.

### Данные о физических лицах
- Долги у приставов.
- Процедура банкротства.
- Миграционные документы.
- Список экстремистов-террористов.
- Розыск МВД.
- Наследственные дела.
- Проверка статуса самозанятого.
- Доверенности.
- Судебные решения.

### Поиск по судебным решениям
- Проблемы с поиском через сервисы судов (часто глючат).
- Необходимость ручного поиска на страницах судов.
- Использование различных сервисов для поиска.

## Ресурсы для изучения
- Государственные реестры и базы данных
- Сервисы судов для поиска решений
- Методики работы с открытыми данными
