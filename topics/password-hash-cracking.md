# Взлом хэшей паролей

## Описание
Техники взлома хэшей паролей с помощью инструментов брутфорса, словарей и радужных таблиц.

## Уровень
Средний

## Примеры использования
- Определение типа хэша.
- Использование Hashcat или John the Ripper.
- Работа со словарями (rockyou.txt).

## Ресурсы для изучения
- [Hashcat Documentation](https://hashcat.net/hashcat/)
- [John the Ripper](https://www.openwall.com/john/)
- Методики взлома различных типов хэшей
