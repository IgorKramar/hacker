# Взлом хэшей паролей

## Описание
Техники взлома хэшей паролей с помощью инструментов брутфорса, словарей и радужных таблиц.

## Уровень
Средний

## Примеры использования
- Определение типа хэша.
- Использование Hashcat или John the Ripper.
- Работа со словарями (rockyou.txt).

## Создание дампа хешей
- Использование Meterpreter для создания дампа хешей паролей пользователей.
- Команда `hashdump` в Meterpreter для извлечения хешей.
- Сохранение дампа хешей для дальнейшего анализа.

## Использование словарей паролей
- Использование списка слов Rockyou.txt для взлома паролей.
- История Rockyou: компания и база данных паролей.
- Использование John the Ripper со списком слов Rockyou для взлома паролей.

## Рекомендации по паролям
- Использование сложных паролей, состоящих из нескольких слов и специальных символов.
- Избегание простых и предсказуемых паролей.
- Регулярная смена паролей для повышения безопасности.

## Ресурсы для изучения
- [Hashcat Documentation](https://hashcat.net/hashcat/)
- [John the Ripper](https://www.openwall.com/john/)
- Методики взлома различных типов хэшей
- Rockyou.txt и другие словари паролей
