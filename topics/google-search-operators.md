# Google поисковые операторы

## Описание
Эффективное использование Google для поиска информации: поисковые операторы, дополнительные инструменты, принципы работы поисковой системы (индексация, обработка запросов).

## Уровень
Основы

## Примеры использования
- Поиск информации о людях, компаниях, событиях.
- Использование операторов для точного поиска (site:, filetype:, intitle:).
- Поиск в определенных доменах и типах файлов.
- Эффективные способы поиска информации в открытых источниках.
