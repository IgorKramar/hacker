# Google поисковые операторы (Google Dorks)

## Описание
Эффективное использование Google для поиска информации: поисковые операторы (Dorks), дополнительные инструменты, принципы работы поисковой системы (индексация, обработка запросов). Google Dorks позволяют находить скрытую информацию, уязвимости и конфиденциальные данные через точные поисковые запросы.

## Уровень
Основы

## Примеры использования
- Поиск информации о людях, компаниях, событиях.
- Использование операторов для точного поиска:
  - `site:` — поиск по содержимому конкретного сайта
  - `filetype:` — поиск документов определенного типа
  - `intitle:` / `allintitle:` — поиск по заголовку страницы
  - `intext:` / `allintext:` — поиск по тексту страницы
  - `inurl:` / `allinurl:` — поиск по URL
  - `-` — исключение слов из результатов
  - `"точная фраза"` — поиск точного совпадения
  - `*` — подстановка любого слова
  - `OR` — логическое "или"
  - `around(N)` — поиск с N словами между терминами
- Комбинирование операторов для уменьшения количества результатов.
- Работа с кэшем Google для доступа к удаленной информации.
- Поиск уязвимостей и конфиденциальных данных.

## Принципы работы Google
- Регистр букв не имеет значения.
- Пробел между словами интерпретируется как логическое разделение.
- Для точного совпадения запрос нужно заключать в кавычки.
- Google склоняет слова автоматически, кроме случаев точного совпадения.
- История запросов и геопозиция влияют на результаты (рекомендуется использовать режим инкогнито).
