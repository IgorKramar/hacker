# Алгоритмы и структуры данных

## Описание
Алгоритмы — последовательности шагов для решения задач, оцениваемые по эффективности через O-сложность. Структуры данных (массивы, стеки, очереди, списки, деревья, графы) организуют данные для оптимизации операций.

## Уровень
Основы

## Примеры использования
- Сортировка массива (например, QuickSort).
- Поиск в графе (DFS, BFS).
