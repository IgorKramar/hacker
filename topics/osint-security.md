# Безопасность и анонимность в OSINT

## Описание
Обеспечение личной безопасности и анонимности при проведении OSINT-расследований: разделение личного и рабочего пространства, использование виртуальных машин, отдельные аккаунты, баланс между комфортом и безопасностью.

## Уровень
Средний

## Примеры использования
- Разделение личного и рабочего пространства.
- Использование виртуальных машин для разделения профилей и задач.
- Заведение отдельных аккаунтов в социальных сетях и мессенджерах.
- Использование средств анонимизации для обхода ограничений.

## Ресурсы для изучения
- Методики обеспечения анонимности
- Best practices для безопасности в OSINT
- Организация рабочего пространства для расследований
