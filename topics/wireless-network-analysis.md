# Анализ беспроводных сетей

## Описание
Анализ Wi-Fi сетей: сбор данных о точках доступа, типах шифрования, географическом расположении и выявление сетей с устаревшими методами защиты.

## Уровень
Средний

## Примеры использования
- Анализ типов шифрования Wi-Fi сетей (WEP, WPA, WPA2, WPA3).
- Выявление сетей с устаревшими методами защиты.
- Географический анализ расположения точек доступа.
- Исследование распространения уязвимых беспроводных сетей.

## Ресурсы для изучения
- Wi-Fi Map и аналогичные инструменты
- Методики анализа беспроводных сетей
- Best practices для защиты Wi-Fi сетей
