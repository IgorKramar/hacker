# Сканирование локальной сети

## Описание
Сканирование локальной сети для обнаружения активных хостов, определения IP-адресов устройств и поиска целевых машин для пентестинга.

## Уровень
Средний

## Примеры использования
- Обнаружение активных хостов в локальной сети.
- Определение IP-адресов устройств (телефоны, компьютеры, уязвимые машины).
- Поиск целевых машин для пентестинга в изолированной сети.
- Проверка связи между машинами с помощью ping.

## Ресурсы для изучения
- Netdiscover и аналогичные инструменты
- Методики сканирования локальных сетей
- Best practices для сетевой разведки
