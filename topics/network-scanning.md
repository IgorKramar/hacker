# Сканирование локальной сети

## Описание
Сканирование локальной сети для обнаружения активных хостов, определения IP-адресов устройств и поиска целевых машин для пентестинга.

## Уровень
Средний

## Примеры использования
- Обнаружение активных хостов в локальной сети.
- Определение IP-адресов устройств (телефоны, компьютеры, уязвимые машины).
- Поиск целевых машин для пентестинга в изолированной сети.
- Проверка связи между машинами с помощью ping.

## Типы IP-адресов
- Локальные IP-адреса принадлежат устройствам внутри сети (телефон, ноутбук, смарт-телевизор).
- Общедоступные IP-адреса принадлежат маршрутизатору и доступны из любой точки мира.

## Результаты сканирования
- Цветовая индикация устройств:
  - Зелёный — устройство реагирует на пинг.
  - Красный — устройство не реагирует.
  - Синий — устройство подключено, но без открытых портов.
- Возможность отображения только устройств с открытыми портами.
- Открытие IP-адресов в веб-браузере для доступа к страницам входа в систему.

## Ресурсы для изучения
- Инструменты для сканирования сети (Netdiscover, Nmap, Angry IP Scanner)
- Методики обнаружения активных хостов
- Best practices для сетевого сканирования
