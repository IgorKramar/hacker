# Брутфорс веб-форм (логин/пароль)

## Описание
Автоматизированные атаки на веб-формы входа с использованием словарей паролей для подбора учетных данных.

## Уровень
Средний

## Примеры использования
- Брутфорс логинов через Burp Suite Intruder.
- Использование словарей (rockyou.txt).
- Настройка атак на веб-формы.

## Ресурсы для изучения
- Burp Suite Intruder документация
- Методики защиты от брутфорса
