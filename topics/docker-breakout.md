# Docker breakout и enumeration

## Описание
Техники выхода из Docker контейнеров (breakout) и enumeration контейнерной среды для получения доступа к хосту.

## Уровень
Продвинутый

## Примеры использования
- Обнаружение Docker окружения.
- Поиск векторов breakout из контейнера.
- Enumeration Docker socket и конфигураций.

## Ресурсы для изучения
- Docker security документация
- Методики Docker breakout
