# Захват и анализ сетевого трафика

## Описание
Захват и анализ сетевого трафика для понимания сетевой активности, отслеживания DNS-запросов, извлечения данных и анализа протоколов.

## Уровень
Средний

## Примеры использования
- Захват трафика с помощью tcpdump.
- Сохранение трафика для дальнейшего анализа.
- Извлечение конкретных данных с помощью tshark.
- Отслеживание DNS-запросов для понимания посещаемых сайтов.
- Анализ протоколов и извлечение паролей из трафика.

## Ресурсы для изучения
- tcpdump и tshark документация
- Wireshark для анализа трафика
- Методики анализа сетевого трафика
