# Социальная инженерия

## Описание
Социальная инженерия — искусство убеждения для получения конфиденциальной информации без использования технических средств взлома. Использует психологические манипуляции и доверие людей для получения доступа к системам и данным.

## Уровень
Средний

## Примеры использования

### Телефонная социальная инженерия
- Звонок от имени банка для получения информации о кредитной карте.
- Имитация технической поддержки для получения паролей.
- Использование доверия людей к официальным организациям.

### Фишинг
- Создание поддельных веб-сайтов для кражи учетных данных.
- Отправка поддельных email-сообщений с вредоносными ссылками.
- Использование социальных сетей для сбора информации.
- SMS-спуфинг для отправки фишинговых сообщений от имени известных организаций.

### Претекстинг
- Создание вымышленного сценария для получения информации.
- Имитация авторитетных лиц или организаций.
- Использование информации из открытых источников (OSINT).

## Инструменты и методы
- Использование ExploitDB для поиска уязвимостей.
- Сбор информации из открытых источников.
- Создание поддельных веб-сайтов и документов.
- Использование психологических техник убеждения.
- SMS-шлюзы (Octopush, Twilio) для подмены SMS-сообщений.

## Защита от социальной инженерии
- Обучение сотрудников распознаванию атак социальной инженерии.
- Проверка подлинности запросов через официальные каналы.
- Ограничение доступа к конфиденциальной информации.
- Использование двухфакторной аутентификации.
- Регулярное обучение вопросам безопасности.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных системах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
- Социальная инженерия против реальных людей является незаконной.
