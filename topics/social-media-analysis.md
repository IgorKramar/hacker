# Анализ социальных сетей в OSINT

## Описание
Анализ профилей в социальных сетях (Facebook, Instagram, ВКонтакте) для сбора информации о людях: поиск профилей по фотографиям, анализ связей, сбор характеризующей информации.

## Уровень
Средний

## Примеры использования
- Поиск профилей ВКонтакте по фотографиям (Find Clown и аналогичные сервисы).
- Анализ профилей Facebook и Instagram.
- Сбор информации о связях и круге общения.
- Рекомендации использовать отдельные аккаунты для сервисов OSINT.

## Инструменты
- **Find Clown**: поиск профилей ВКонтакте по фотографиям.
- Специализированные инструменты для анализа различных социальных сетей.

## Ресурсы для изучения
- Инструменты для анализа социальных сетей
- Методики поиска профилей по фотографиям
- Best practices для работы с социальными сетями в OSINT
