# Неправильная конфигурация CORS

## Описание
CORS (Cross-Origin Resource Sharing) — механизм совместного использования ресурсов разных источников, позволяющий веб-страницам запрашивать ресурсы с других доменов. Неправильная конфигурация CORS может привести к утечкам данных и междоменным атакам.

## Уровень
Средний

## Примеры использования

### Механизм CORS
- Механизм совместного использования ресурсов разных источников.
- Позволяет веб-страницам запрашивать ресурсы с других доменов.
- Потенциал для междоменных атак при некорректной настройке политики.

### Процесс атаки
- Исполнитель угрозы отправляет вредоносный скрипт жертве через веб-сайт.
- Скрипт устанавливает соединение с другим веб-сайтом, собирая информацию.
- Информация передаётся от жертвы на веб-сайт исполнителя угрозы.
- Скрипт использует учётные данные для доступа к веб-сайту.
- Может отправлять учётные данные по электронной почте или другим способом.

### Неправильная конфигурация
- Разрешение запросов с любых источников (`Access-Control-Allow-Origin: *`).
- Отсутствие проверки источников запросов.
- Разрешение небезопасных методов (PUT, DELETE) для всех источников.
- Отсутствие проверки заголовков и credentials.

## Типы уязвимостей

### Слишком широкие разрешения
- Разрешение доступа с любых доменов.
- Отсутствие ограничений на методы и заголовки.
- Разрешение отправки credentials с любых источников.

### Отсутствие валидации
- Приложение не проверяет источник запроса.
- Автоматическое разрешение всех CORS-запросов.
- Отсутствие whitelist для разрешенных доменов.

## Защита от неправильной конфигурации CORS
- Использование whitelist для разрешенных источников.
- Ограничение методов и заголовков для CORS-запросов.
- Правильная настройка credentials и заголовков.
- Валидация источников запросов.
- Использование строгих политик CORS.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных приложениях или с явного разрешения.
- Понимание юридических последствий незаконного использования.
