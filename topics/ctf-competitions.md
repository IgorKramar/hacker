# CTF (Capture The Flag) соревнования

## Описание
Capture The Flag (CTF) — соревнования по кибербезопасности, где участники решают задачи на поиск уязвимостей, взлом систем и получение флагов для практики навыков этичного хакерства.

## Уровень
Средний

## Примеры использования
- Участие в CTF-соревнованиях для практики пентестинга.
- Прохождение CTF-машин на платформах (VulnHub, TryHackMe, HackTheBox).
- Получение флагов (user.txt, root.txt) для подтверждения успешного взлома.
- Развитие навыков этичного хакерства в безопасной среде.

## Ресурсы для изучения
- [VulnHub](https://www.vulnhub.com/)
- [TryHackMe](https://tryhackme.com/)
- [HackTheBox](https://www.hackthebox.com/)
- CTF методики и writeups
