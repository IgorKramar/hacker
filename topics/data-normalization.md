# Нормализация данных

## Описание
Нормализация данных в базах данных: нормальные формы (1NF, 2NF, 3NF) для устранения избыточности и обеспечения целостности.

## Уровень
Средний

## Примеры использования
- Приведение базы данных к нормальным формам.
- Устранение дублирования данных.
- Оптимизация структуры таблиц.

## Ресурсы для изучения
- Книги по проектированию баз данных
- Материалы по нормализации
