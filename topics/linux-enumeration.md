# Перечисление систем Linux (Linux Enumeration)

## Описание
Перечисление систем Linux — ключевой аспект повышения привилегий, включающий сбор информации о системе, версии ядра, архитектуре, процессах, пользователях, сетях и конфигурации для поиска уязвимостей.

## Уровень
Средний

## Ручное перечисление

### Анализ системы
- Использование `uname -a` для определения версии ядра и архитектуры системы.
- Важность знания архитектуры процессора и количества потоков.
- Сужение круга поиска эксплойтов на основе информации о процессоре.

### Анализ процессов
- Использование `ps ax` для просмотра запущенных служб.
- Анализ процессов пользователя root.
- Примеры запущенных служб: nginx, apache, cron.

### Перечисление пользователей
- Определение идентификатора пользователя и группы.
- Использование команды `sudo -l` для проверки привилегий.
- Возможность запуска команд от имени root без пароля.

### Анализ конфиденциальных файлов
- Проверка доступа к конфиденциальным файлам (`/etc/passwd`, `/etc/shadow`).
- Просмотр истории команд для поиска паролей и важной информации.
- Анализ изменений файлов за последние минуты.

### Перечисление сетей
- Важность понимания IP-архитектуры и открытых портов.
- Команды для получения информации о сети: `ifconfig`, `ip a`.
- Использование `route` для проверки маршрутов.
- Команда `netstat` для определения открытых портов и коммуникаций.
- Анализ таблиц ARP для взаимодействия с другими устройствами.

### Поиск паролей
- Использование `grep` для поиска паролей в файлах.
- Настройка цвета для выделения найденных строк.
- Сужение поиска с помощью дополнительных условий (например, "password").
- Поиск паролей в домашней папке и конфигурационных файлах.

### Поиск SSH-ключей
- Поиск SSH-ключей на компьютере для получения доступа к другим пользователям или машинам.
- Использование команды для поиска авторизованных ключей или идентификатора RSA.
- Пример поиска: "backup super secret keys id rsa".

## Автоматизированные инструменты

### LinPEAS
- Скрипт для повышения привилегий в Linux с цветовой координацией.
- Выполняет автоматическое перечисление и выявляет уязвимости.
- Анализ процессов, cron, сервисов, сетевой информации.
- Проверка информации о пользователях и входах в систему.

### Linuxprivchecker.py
- Инструмент на Python для анализа привилегий.
- Регулярные обновления и актуальность.

### LSE (Linux Smart Enumeration)
- Популярный инструмент с регулярными обновлениями.
- Почти 2000 звёзд на GitHub.

### linenum.sh
- Скрипт для поиска эксплойтов в Linux.
- Быстрое обнаружение потенциальных уязвимостей.
