# Сканирование портов (port scanning)

## Описание
Обнаружение открытых портов и сервисов на целевой системе с помощью инструментов сканирования портов.

## Уровень
Основы

## Примеры использования
- Базовое сканирование портов с Nmap.
- Определение версий сервисов.
- Обнаружение активных хостов.

## Ресурсы для изучения
- [Nmap Documentation](https://nmap.org/book/)
- TryHackMe комнаты по сканированию портов
