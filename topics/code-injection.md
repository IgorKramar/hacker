# Code Injection

## Описание
Code Injection — уязвимость, позволяющая злоумышленникам выполнять произвольный код на сервере через неправильную обработку пользовательского ввода, что может привести к полному доступу к сайту, файлам и базе данных.

## Уровень
Продвинутый

## Примеры использования
- Внедрение ERB-кода в шаблоны (Ruby on Rails).
- Выполнение произвольных команд на сервере.
- Доступ к файлам и базе данных через code injection.
- Получение полного контроля над приложением.

## Ресурсы для изучения
- Методики защиты от code injection
- Правильная обработка пользовательского ввода
- Использование безопасных методов рендеринга
