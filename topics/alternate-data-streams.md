# Альтернативные потоки данных (Alternate Data Streams)

## Описание
Альтернативные потоки данных (ADS) — функция файловой системы NTFS, позволяющая хранить дополнительную информацию внутри файлов. Могут быть использованы для скрытия информации, которая может быть использована для получения доступа к привилегированным учетным записям или выполнения вредоносных действий.

## Уровень
Средний

## Примеры использования

### Сокрытие информации
- Альтернативные потоки данных могут быть использованы для скрытия информации внутри файлов.
- Могут быть использованы для сокрытия информации, которая может быть использована для получения доступа к привилегированным учетным записям.
- Скрытие вредоносных файлов и скриптов.

### Процесс создания ADS
1. Создание основного файла.
2. Добавление альтернативного потока данных к файлу.
3. Сохранение информации в альтернативном потоке.
4. Доступ к информации через альтернативный поток.

## Типы использования

### Скрытие файлов
- Использование ADS для скрытия файлов от обычных методов обнаружения.
- Скрытие вредоносных скриптов и исполняемых файлов.

### Скрытие данных
- Использование ADS для скрытия конфиденциальных данных.
- Скрытие учетных данных и другой чувствительной информации.
