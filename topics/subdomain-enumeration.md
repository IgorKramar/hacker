# Поиск поддоменов

## Описание
Поиск поддоменов целевого домена с использованием wordlist для обнаружения скрытых сервисов, административных панелей и дополнительных векторов атаки.

## Уровень
Средний

## Примеры использования
- Использование wordlist для поиска поддоменов.
- Брутфорс поддоменов с помощью инструментов (Gobuster, ffuf).
- Использование wordlist из коллекций (SecLists, GitHub).
- Обнаружение скрытых сервисов и административных панелей.

## Ресурсы для изучения
- Gobuster, ffuf для поиска поддоменов
- Wordlists: SecLists, DNS wordlists
- Методики enumeration поддоменов
