# Взлом passphrase ключей

## Описание
Взлом passphrase защищенных SSH-ключей с помощью инструментов брутфорса для получения доступа к серверам.

## Уровень
Средний

## Примеры использования
- Использование John the Ripper для взлома passphrase.
- Работа со словарями паролей.
- Восстановление доступа через защищенные ключи.

- John the Ripper документация
- Методики взлома passphrase
