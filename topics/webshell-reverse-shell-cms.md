# Внедрение webshell/reverse shell в CMS

## Описание
Техники внедрения webshell или reverse shell в системы управления контентом (CMS) для получения удаленного доступа к серверу.

## Уровень
Продвинутый

## Примеры использования
- Генерация payload с msfvenom.
- Загрузка webshell через уязвимости загрузки файлов.
- Настройка reverse shell соединения.

## Ресурсы для изучения
- Metasploit Framework документация
- Методики эксплуатации CMS
