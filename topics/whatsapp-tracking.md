# Отслеживание WhatsApp аккаунтов

## Описание
Методы отслеживания WhatsApp аккаунтов и сбора информации о пользователях без взлома телефона или компьютера жертвы. Включает получение IP-адресов через звонки, определение местоположения, сбор личной информации из открытых источников и проверку утечек данных.

## Уровень
Продвинутый

## Примеры использования

### Получение IP-адреса через звонок
- Установка настольного приложения WhatsApp и программы Wireshark для анализа сетевого трафика.
- Сохранение номера телефона на телефоне и установка WhatsApp на рабочий стол.
- Получение локального IP-адреса через командную строку.
- Фильтрация трафика по локальному IP-адресу в Wireshark.
- Звонок человеку и получение его публичного IP-адреса из сетевого трафика.
- Использование IP-адреса для определения местоположения (страна и город).

### Сбор личной информации из WhatsApp
- Использование веб-сайта WhatsApp для получения информации о номере телефона:
  - Имя пользователя
  - Фотография профиля
  - Биография
- Обратный поиск изображений в Google для получения дополнительной информации по фотографии профиля.

### Поиск в социальных сетях
- Поиск информации о человеке в LinkedIn по имени и другим данным.
- Определение даты рождения и университета, где учился человек.
- Возможность запроса физического адреса у правительства на основе полученной информации.

### Проверка утечек данных
- Проверка адреса электронной почты на сайтах утечек данных (например, Intelligence X).
- Поиск утечек данных, содержащих физические адреса.
- Загрузка информации об утечке данных для получения дополнительной информации.

### Использование инструментов для поиска
- Использование программ для поиска информации в файлах (например, Agent Transact).
- Поиск по адресу электронной почты в файлах на рабочем столе.
- Получение полного адреса, дня рождения, адреса электронной почты и другой информации.

## План сбора информации
1. Позвонить в целевую учётную запись.
2. Узнать IP-адрес через анализ сетевого трафика.
3. Определить страну и город по IP-адресу.
4. Получить полное имя и дату рождения из открытых источников.
5. Собрать дополнительную информацию из утечек данных.

## Защита от отслеживания
- Использование VPN для скрытия реального IP-адреса.
- Ограничение доступа к личной информации в настройках WhatsApp.
- Регулярная проверка утечек данных и смена паролей.
- Осторожность при предоставлении личной информации в социальных сетях.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных аккаунтах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
- Несанкционированное отслеживание других людей является незаконным.
