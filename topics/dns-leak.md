# DNS утечки (DNS Leak)

## Описание
DNS утечки — проблема безопасности, когда DNS-запросы отправляются через обычный DNS-сервер вместо защищенного прокси, раскрывая реальный IP-адрес и местоположение пользователя.

## Уровень
Средний

## Примеры использования
- Проверка DNS утечек через специальные сервисы (DNS Leak Test).
- Настройка SOCKS5 для предотвращения утечек DNS.
- Использование Tor для защиты от DNS утечек.
- Проверка конфигурации прокси на корректность работы.

## Ресурсы для изучения
- [DNS Leak Test](https://www.dnsleaktest.com/)
- Методики защиты от DNS утечек
- Документация по настройке DNS в Tor
