# Поиск конфигурационных файлов

## Описание
Поиск конфигурационных файлов (wp-config.php, .backup, .env) на веб-серверах для обнаружения данных авторизации, API-ключей и другой чувствительной информации.

## Уровень
Средний

## Примеры использования
- Поиск wp-config.php в WordPress приложениях.
- Обнаружение .backup файлов с данными авторизации.
- Чтение .env файлов с конфигурацией и ключами.
- Использование параметров страницы для чтения файлов.

## Ресурсы для изучения
- Методики поиска конфигурационных файлов
- OWASP рекомендации по защите конфигурационных файлов
- Инструменты для автоматического поиска (Gobuster, Dirbuster)
