# Модификация прав файлов (chmod, chown)

## Описание
Использование команд chmod и chown для модификации прав доступа к файлам и повышения привилегий.

## Уровень
Продвинутый

## Примеры использования
- Добавление SUID к бинарникам (chmod +s).
- Изменение владельца файлов (chown).
- Эксплуатация неправильных прав доступа.

## Ресурсы для изучения
- Linux файловые права документация
- Методики privilege escalation через права файлов
