# Чтение приватных ключей (SSH)

## Описание
Обнаружение и чтение приватных SSH-ключей на целевой системе для получения доступа к другим серверам.

## Уровень
Средний

## Примеры использования
- Поиск SSH-ключей в домашних директориях.
- Чтение приватных ключей из конфигурационных файлов.
- Использование найденных ключей для доступа.

## Ресурсы для изучения

- Методики поиска ключей
