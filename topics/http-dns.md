# HTTP-запросы и DNS

## Описание
HTTP-запросы (GET, POST, PUT, DELETE), протокол DNS, корневые серверы, кэширование и процесс разрешения доменных имен.

## Уровень
Основы

## Примеры использования
- Отправка различных типов HTTP-запросов.
- Понимание работы DNS и разрешения имен.
- Анализ DNS-запросов и кэширования.

## Ресурсы для изучения

- Документация по DNS
