# Геолокация и анализ местоположения

## Описание
Геолокация в OSINT: выявление и идентификация ориентиров на фотографиях и видео, сопоставление ориентиров для выдвижения гипотез, проверка гипотез через изучение карт и спутниковых снимков.

## Уровень
Средний

## Примеры использования
- Идентификация ориентиров на фотографиях (здания, вывески, природные объекты).
- Сопоставление ориентиров для определения местоположения.
- Использование карт и спутниковых снимков для проверки гипотез.
- Анализ геолокационных данных из метаданных.
- Определение местоположения по визуальным признакам.

## Ресурсы для изучения
- Google Maps, Яндекс.Карты
- Спутниковые снимки (Google Earth)
- Методики геолокации по фотографиям
