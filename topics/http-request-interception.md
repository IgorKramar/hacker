# Перехват и модификация HTTP-запросов

## Описание
Перехват, анализ и модификация HTTP-запросов и ответов для тестирования безопасности и эксплуатации уязвимостей.

## Уровень
Продвинутый

## Примеры использования
- Использование Burp Suite Proxy.
- Модификация POST/GET параметров.
- Анализ заголовков и cookies.

## Ресурсы для изучения
- [Burp Suite Documentation](https://portswigger.net/burp/documentation)
- OWASP Web Security Testing Guide
