# Анализ бинарников и path hijacking

## Описание
Анализ бинарных файлов для понимания их поведения и эксплуатация path hijacking для повышения привилегий.

## Уровень
Продвинутый

## Примеры использования
- Анализ бинарников с помощью strings и objdump.
- Понимание работы бинарника.
- Создание кастомных скриптов для path hijacking.
