# Брутфорс паролей

## Описание
Автоматизированный подбор паролей для различных сервисов (SSH, FTP, веб-формы авторизации) с использованием словарей и инструментов брутфорса.

## Уровень
Средний

## Примеры использования
- Подбор паролей для SSH с помощью Hydra.
- Брутфорс FTP серверов.
- Атака на веб-формы авторизации.
- Использование словарей (rockyou.txt) для брутфорса.

## Ресурсы для изучения
- Hydra документация
- Методики брутфорса паролей
- Wordlists для брутфорса
