# Сбор информации о человеке

## Описание
Быстрый сбор базовой информации о человеке: идентификация, сбор персональных данных, поиск аккаунтов и контактов (телефоны, почты, никнеймы, социальные сети), анализ связей, поиск по фото, характеризующая информация. Подход должен быть гибким и адаптивным для эффективности.

## Уровень
Средний

## Базовый набор данных для идентификации
- ФИО (полное имя)
- Дата рождения
- Телефон
- Email
- Адрес

## Примеры использования

### Идентификация человека
- Использование уникальных идентификаторов и комбинаций данных.
- Точная идентификация для избежания путаницы с однофамильцами.
- Подготовка шаблонной таблицы для систематизации данных.

### Источники первичной информации
- Агрегаторы утечек данных (сервисы OSINT, боты в Telegram).
- Уточнение запросов с помощью дополнительных операторов.
- Поиск в базах данных утечек.

### Поиск через поисковики
- Использование ФИО в кавычках для точного поиска: `"Имя Фамилия"`.
- Фильтрация результатов с помощью оператора минус: `"Имя Фамилия" -стоматолог -хирург`.
- Поиск по региональным доменам: `site:*.fr "Pavel Durov"`.
- Использование операторов intitle и intext для конкретизации: `intitle:"Имя Фамилия" intext:биография`.

### Поиск аккаунтов и активностей
- Поиск по никнеймам в URL: `inurl:никнейм`.
- Поиск по электронным почтам и социальным сетям.
- Использование уникальных имен и комбинаций для поиска страниц.

### Поиск документов
- Поиск документов, в которых упоминается человек.
- Запросы по фамилии, имени и отчеству.
- Для публичных лиц - проверка на разных языках.

### Специализированные инструменты
- **My Name Up**: поиск по никнейму, ФИО в разных транскрипциях, нейтральные результаты.
- **Name Mint**: формирование комбинаций из имени и фамилии, проверка логинов и email адресов.

### Дополнительные техники
- Анализ связей (родственники, круг общения, деловые контакты).
- Поиск по фото для обнаружения связанных аккаунтов и упоминаний.
- Обратный поиск изображений в Google для получения дополнительной информации.
- Сбор характеризующей информации (увлечения, интересы, образ жизни).
- Использование прокси для обхода блокировок и подмены геопозиции.
- Поиск информации в LinkedIn по имени и другим данным.
- Определение даты рождения и университета из открытых источников.

## Ресурсы для изучения
- Методики сбора персональных данных
- Инструменты для поиска по фото
- Агрегаторы утечек данных
- My Name Up, Name Mint и аналогичные инструменты
- Best practices для этичного сбора информации
