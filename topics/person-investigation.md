# Сбор информации о человеке

## Описание
Сбор персональных данных о человеке: подготовка шаблонной таблицы, поиск аккаунтов и контактов (телефоны, почты, никнеймы, социальные сети), анализ связей, поиск по фото, характеризующая информация.

## Уровень
Средний

## Примеры использования
- Поиск аккаунтов в социальных сетях по никнейму или фото.
- Сбор контактных данных (телефоны, email, адреса).
- Анализ связей (родственники, круг общения, деловые контакты).
- Поиск по фото для обнаружения связанных аккаунтов и упоминаний.
- Сбор характеризующей информации (увлечения, интересы, образ жизни).

## Ресурсы для изучения
- Методики сбора персональных данных
- Инструменты для поиска по фото
- Best practices для этичного сбора информации
