# SQL Injection

## Описание
SQL Injection — уязвимость, позволяющая злоумышленнику внедрить код в SQL-запрос, изменяя логику запроса и получая доступ к конфиденциальной информации, включая данные всех пользователей.

## Уровень
Средний

## Примеры использования
- Добавление условия `OR 1=1` для обхода аутентификации.
- Использование оператора `UNION` для объединения запросов.
- Подбор столбцов и типов полей для успешного выполнения запроса.
- Извлечение хэшей паролей и конфиденциальных данных.
- Демонстрация SQL-инъекций в строках поиска и регистрационных формах.
- Перехват поискового запроса и отправка его на ретранслятор.
- Получение всех узлов из базы данных через SQL-инъекцию.
- Экранирование специальных символов для успешной вставки запроса.
- Получение доступа к парольным фразам через SQL-инъекцию.

## Инструменты для эксплуатации
- **SQLMap**: автоматизация обнаружения и использования SQL-инъекций.
- Дактилоскопия базы данных для определения типа и версии.
- Извлечение данных из таблиц базы данных, не используемых веб-приложением.
