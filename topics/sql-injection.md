# SQL Injection

## Описание
SQL Injection — уязвимость, позволяющая злоумышленнику внедрить код в SQL-запрос, изменяя логику запроса и получая доступ к конфиденциальной информации, включая данные всех пользователей.

## Уровень
Средний

## Примеры использования
- Добавление условия `OR 1=1` для обхода аутентификации.
- Использование оператора `UNION` для объединения запросов.
- Подбор столбцов и типов полей для успешного выполнения запроса.
- Извлечение хэшей паролей и конфиденциальных данных.

## Ресурсы для изучения
- [OWASP SQL Injection](https://owasp.org/www-community/attacks/SQL_Injection)
- Использование параметризованных запросов
- Методики защиты от SQL Injection
