# Meterpreter

## Описание
Meterpreter — продвинутый payload для Metasploit Framework, предоставляющий интерактивный командный интерфейс для выполнения команд на скомпрометированной системе. Позволяет получать доступ к файлам, создавать дампы хешей, выполнять команды и управлять системой.

## Уровень
Продвинутый

## Примеры использования

### Получение доступа
- Запуск эксплойта через Metasploit и получение приглашения от Meterpreter.
- Вход в систему Windows после успешной эксплуатации уязвимости.
- Проверка доступных команд для управления системой.

### Работа с системой
- Выполнение команд на скомпрометированной системе.
- Доступ к файлам и директориям.
- Создание дампа хешей паролей пользователей.
- Управление процессами и сервисами.

### Дополнительные возможности
- Захват скриншотов экрана.
- Запись нажатий клавиш (keylogging).
- Перехват сетевого трафика.
- Создание туннелей для доступа к внутренней сети.

## Команды Meterpreter

### Общие команды
- `hashdump` — создание дампа хешей паролей.
- `shell` — получение интерактивной командной строки.
- `screenshot` — захват скриншота экрана.
- `keyscan_start` / `keyscan_stop` — запись нажатий клавиш.
- `migrate` — миграция процесса для скрытности.
- `sysinfo` — получение информации о системе.
- `getuid` — определение учетной записи пользователя.
- `help` — просмотр всех доступных команд.

### Команды для Android устройств
- `check_root` — проверка рутирования устройства.
- `geolocate` — определение физического местоположения устройства.
- `dump_calllog` — извлечение истории вызовов.
- `dump_contacts` — извлечение контактов.
- `dump_sms` — извлечение SMS-сообщений.
- `webcam_list` — просмотр доступных камер.
- `webcam_snap` — захват фотографий с камеры.
- `record_mic` — запись звука через микрофон (например, `record_mic -d 10` для 10 секунд).

## Интеграция с Metasploit
- Использование Meterpreter payload в различных эксплойтах Metasploit.
- Автоматизация атак с помощью Meterpreter.
- Пост-эксплуатация после получения доступа к системе.

## Защита от Meterpreter
- Установка обновлений безопасности для защиты от уязвимостей.
- Использование антивирусов и систем обнаружения вторжений.
- Мониторинг сетевой активности на предмет подозрительных соединений.
- Регулярное обновление систем для защиты от известных эксплойтов.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных системах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
