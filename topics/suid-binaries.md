# Поиск SUID-бинарников

## Описание
Поиск и эксплуатация SUID (Set User ID) бинарников для повышения привилегий в Linux системах.

## Уровень
Средний

## Примеры использования
- Поиск SUID-файлов с помощью find.
- Анализ SUID-бинарников на уязвимости.
- Эксплуатация для получения root-доступа.

## Ресурсы для изучения
- Linux privilege escalation методики
- GTFOBins для SUID эксплуатации
