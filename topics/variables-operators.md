
# Переменные и операторы

## Описание

**Переменные** и **операторы** являются фундаментом любого программного кода. Переменная — это именованная область памяти, в которой хранится значение определённого типа данных. Оператор — это символ или набор символов, который указывает процессору на выполнение определённой операции над данными. В совокупности переменные и операторы позволяют создавать логику программ, управлять потоком исполнения кода и обрабатывать информацию[1][2].

Условные операторы (if, switch) позволяют выполнять разные блоки кода в зависимости от условий, а циклы (for, while, do-while) обеспечивают повторение операций нужное количество раз[3][4]. Вместе эти конструкции составляют основу структурированного программирования и позволяют реализовать любые алгоритмы.

## Переменные

### Что такое переменная

**Переменная** — это именованная область в памяти компьютера, предназначенная для хранения данных. Каждая переменная имеет три ключевых характеристики[2]:

- **Имя** — уникальный идентификатор, используемый для обращения к переменной (например, `age`, `name`, `sum`)
- **Тип** — определяет, какие значения может хранить переменная и какие операции над ней можно выполнять
- **Значение** — актуальные данные, хранящиеся в переменной в данный момент

### Типы данных

Каждый язык программирования предоставляет набор встроенных типов данных. Основные типы, встречающиеся во всех современных языках[1][5][6]:

| Тип данных | Описание | Примеры | Размер в памяти |
|---|---|---|---|
| **Целочисленный (int)** | Целые числа без дробной части | -5, 0, 100, 42 | 4 байта (обычно) |
| **Вещественный (float)** | Числа с плавающей точкой, одинарная точность | 3.14, -0.5, 2.71828 | 4 байта |
| **Вещественный (double)** | Числа с плавающей точкой, двойная точность, более точный | 3.14159265, -0.00001 | 8 байт |
| **Символьный (char)** | Одиночный символ | 'A', 'б', '1', '@' | 1-2 байта |
| **Строковый (string)** | Последовательность символов | "Hello", "Привет мир" | Переменный размер |
| **Логический (bool)** | Истина или ложь | true, false | 1 байт |
| **Массив** | Упорядоченная коллекция элементов одного типа | [7][8][9], ['a', 'b'] | Зависит от содержимого |

### Объявление и инициализация переменных

Способ объявления переменной зависит от языка программирования, но принцип остаётся общим. В языках со статической типизацией тип указывается явно[2]:

```cpp
int age = 25;           // Целое число
float height = 1.75;    // Вещественное число
char grade = 'A';       // Символ
bool is_student = true; // Логическое значение
```

В языках с динамической типизацией тип определяется автоматически[1][2]:

```python
age = 25                    # int определён автоматически
name = "Иван"              # string определён автоматически
height = 1.75              # float определён автоматически
is_active = True            # bool определён автоматически
```

### Статическая vs динамическая типизация

**Статическая типизация** — тип переменной определяется при компиляции и не может изменяться. Используется в C, C++, Java, C#[1][2]:

- Преимущества: обнаружение ошибок типов на этапе компиляции, выше производительность
- Недостатки: меньше гибкости, больше кода

**Динамическая типизация** — тип определяется во время выполнения программы. Используется в Python, JavaScript, PHP[1][2]:

- Преимущества: большая гибкость, меньше кода, быстрая разработка
- Недостатки: больше ошибок могут проявиться в runtime, ниже производительность

### Область видимости переменных

**Область видимости (scope)** — часть программы, в пределах которой переменная доступна и может быть использована[2]:

- **Глобальная область видимости** — переменная доступна во всей программе
- **Локальная область видимости** — переменная доступна только внутри функции или блока кода
- **Блочная область видимости** — переменная доступна только внутри блока (часто { })

Пример:

```cpp
int global_var = 10;  // Глобальная переменная

void function() {
    int local_var = 5;  // Локальная переменная
    {
        int block_var = 3;  // Видна только в этом блоке
        cout << block_var;  // OK
    }
    // cout << block_var;  // ОШИБКА: block_var не видна здесь
}
```

## Операторы

### Арифметические операторы

**Арифметические операторы** используются для выполнения математических операций над числовыми значениями[10][11]:

| Оператор | Название | Пример | Результат |
|---|---|---|---|
| `+` | Сложение | 5 + 3 | 8 |
| `-` | Вычитание | 5 - 3 | 2 |
| `*` | Умножение | 5 * 3 | 15 |
| `/` | Деление | 6 / 2 | 3 |
| `%` | Остаток от деления (модуль) | 7 % 3 | 1 |
| `++` | Инкремент (увеличение на 1) | i++ | увеличивает i |
| `--` | Декремент (уменьшение на 1) | i-- | уменьшает i |

Пример использования:

```java
int a = 10;
int b = 3;

int sum = a + b;       // 13
int diff = a - b;      // 7
int prod = a * b;      // 30
int quot = a / b;      // 3 (целочисленное деление)
int remainder = a % b; // 1

a++;  // a становится 11
b--;  // b становится 2
```

### Операторы присваивания

**Операторы присваивания** используются для присвоения значения переменной[10]:

| Оператор | Эквивалент | Пример |
|---|---|---|
| `=` | Присваивание | x = 5 |
| `+=` | x = x + | x += 5 (эквивалентно x = x + 5) |
| `-=` | x = x - | x -= 3 (эквивалентно x = x - 3) |
| `*=` | x = x * | x *= 2 (эквивалентно x = x * 2) |
| `/=` | x = x / | x /= 4 (эквивалентно x = x / 4) |
| `%=` | x = x % | x %= 3 (эквивалентно x = x % 3) |

Пример:

```java
int x = 0;
x += 10; // x = 0 + 10 => x = 10
x -= 5;  // x = 10 - 5 => x = 5
x *= 5;  // x = 5 * 5 => x = 25
x /= 5;  // x = 25 / 5 => x = 5
x %= 3;  // x = 5 % 3 => x = 2
```

### Операторы сравнения

**Операторы сравнения** используются для сравнения двух значений и возвращают логическое значение true (истина) или false (ложь)[10][12][13]:

| Оператор | Название | Пример | Результат |
|---|---|---|---|
| `==` | Равно | 5 == 5 | true |
| `!=` | Не равно | 5 != 3 | true |
| `>` | Больше чем | 5 > 3 | true |
| `<` | Меньше чем | 5 < 3 | false |
| `>=` | Больше или равно | 5 >= 5 | true |
| `<=` | Меньше или равно | 3 <= 5 | true |

Пример использования:

```java
int a = 5;
int b = 3;

boolean result1 = a == b;  // false
boolean result2 = a != b;  // true
boolean result3 = a > b;   // true
boolean result4 = a >= b;  // true
boolean result5 = a < b;   // false
boolean result6 = a <= b;  // false
```

### Логические операторы

**Логические операторы** используются для комбинирования логических выражений и создания сложных условий[10][11][12]:

| Оператор | Название | Значение | Пример |
|---|---|---|---|
| `&&` | Логическое И (AND) | Истина только если оба операнда истинны | (a > 0) && (b > 0) |
| `\|\|` | Логическое ИЛИ (OR) | Истина если хотя бы один операнд истинен | (a > 0) \|\| (b > 0) |
| `!` | Логическое НЕ (NOT) | Инвертирует логическое значение | !(a > 0) |

Пример использования:

```java
int age = 25;
int score = 85;

// Оператор &&
boolean both_conditions = (age > 18) && (score > 80);  // true (оба условия верны)

// Оператор ||
boolean either_condition = (age < 18) || (score > 80);  // true (второе условие верно)

// Оператор !
boolean not_condition = !(age < 18);  // true (инвертируем false в true)
```

### Приоритет и порядок выполнения операторов

При вычислении выражения с несколькими операторами они выполняются в определённом порядке, называемом **приоритетом** (precedence)[14][15][16]:

| Приоритет | Операторы | Направление |
|---|---|---|
| 1 (Высший) | Скобки `()` | - |
| 2 | Унарный минус `-`, логическое НЕ `!` | Справа налево |
| 3 | Умножение `*`, деление `/`, модуль `%` | Слева направо |
| 4 | Сложение `+`, вычитание `-` | Слева направо |
| 5 | Операторы сравнения `<`, `>`, `<=`, `>=` | Слева направо |
| 6 | Операторы равенства `==`, `!=` | Слева направо |
| 7 | Логическое И `&&` | Слева направо |
| 8 | Логическое ИЛИ `\|\|` | Слева направо |
| 9 (Низший) | Присваивание `=`, `+=`, `-=` | Справа налево |

Примеры:

```
2 + 3 * 4 = 2 + 12 = 14         (умножение до сложения)
(2 + 3) * 4 = 5 * 4 = 20        (скобки имеют высший приоритет)
4 + 4 / 2 = 4 + 2 = 6           (деление до сложения)
```

Используя скобки, можно изменить порядок выполнения операций[15].

## Условные операторы

Условные операторы позволяют программе выполнять разные блоки кода в зависимости от выполнения или невыполнения определённых условий.

### Оператор if

**Оператор if** — самый основной условный оператор. Он выполняет блок кода, если условие истинно (true)[17][4]:

```java
if (условие) {
    // Код выполняется, если условие истинно
}
```

Пример:

```java
int x = 10;

if (x > 5) {
    System.out.println("x больше 5");
}
```

### Конструкция if-else

**if-else** позволяет указать альтернативный блок кода, который выполнится, если условие ложно (false)[17][4]:

```java
if (условие) {
    // Код выполняется, если условие истинно
} else {
    // Код выполняется, если условие ложно
}
```

Пример:

```java
int age = 15;

if (age >= 18) {
    System.out.println("Вы совершеннолетний");
} else {
    System.out.println("Вы несовершеннолетний");
}
```

### Цепочка if-else if-else

Для проверки нескольких условий используется **if-else if-else**[17][4]:

```java
if (условие1) {
    // Код если условие1 истинно
} else if (условие2) {
    // Код если условие1 ложно и условие2 истинно
} else if (условие3) {
    // Код если условие1 и условие2 ложны и условие3 истинно
} else {
    // Код если все условия ложны
}
```

Пример:

```java
int a = 20;

if (a == 10) {
    System.out.println("a = 10");
} else if (a == 15) {
    System.out.println("a = 15");
} else if (a == 20) {
    System.out.println("a = 20");
} else {
    System.out.println("Ничего из перечисленного");
}
// Вывод: a = 20
```

### Оператор switch

**Оператор switch** используется для выбора одного из нескольких вариантов на основе значения выражения. Он более эффективен, когда нужно сравнивать переменную с несколькими фиксированными значениями[17][4]:

```java
switch (выражение) {
    case значение1:
        // Код если выражение == значение1
        break;
    case значение2:
        // Код если выражение == значение2
        break;
    default:
        // Код если ни один case не совпадает
}
```

Пример:

```java
int day = 3;
String dayName;

switch (day) {
    case 1:
        dayName = "Понедельник";
        break;
    case 2:
        dayName = "Вторник";
        break;
    case 3:
        dayName = "Среда";
        break;
    case 4:
        dayName = "Четверг";
        break;
    case 5:
        dayName = "Пятница";
        break;
    default:
        dayName = "Выходной";
}

System.out.println(dayName);
// Вывод: Среда
```

Важно использовать `break` для выхода из switch, иначе выполнение продолжится до следующего case.

### Тернарный условный оператор

**Тернарный оператор** (`? :`) — компактный способ записать if-else на одну строку[17]:

```java
result = условие ? значение_если_истинно : значение_если_ложно;
```

Пример:

```java
int age = 20;
String status = (age >= 18) ? "Взрослый" : "Подросток";
System.out.println(status);
// Вывод: Взрослый
```

## Циклы

Циклы позволяют выполнять блок кода многократно. Существует несколько типов циклов, каждый из которых подходит для разных ситуаций.

### Цикл for

**Цикл for** — цикл с известным числом повторений. Он состоит из инициализации, условия и модификации переменной[3][18][19]:

```java
for (инициализация; условие; модификация) {
    // Тело цикла
}
```

Процесс выполнения:

1. **Инициализация** — переменная-счётчик устанавливается на начальное значение
2. **Условие** — проверяется истинность условия; если true, выполняется тело
3. **Тело цикла** — выполняется код внутри цикла
4. **Модификация** — счётчик изменяется (обычно увеличивается)
5. Возвращаемся к шагу 2

Пример:

```java
for (int i = 0; i < 5; i++) {
    System.out.println("i = " + i);
}
// Вывод:
// i = 0
// i = 1
// i = 2
// i = 3
// i = 4
```

### Цикл while

**Цикл while** — цикл с предусловием. Тело цикла выполняется, пока условие истинно[3][18][19]:

```java
while (условие) {
    // Тело цикла
}
```

Пример:

```java
int i = 0;

while (i < 5) {
    System.out.println("i = " + i);
    i++;
}
// Вывод:
// i = 0
// i = 1
// i = 2
// i = 3
// i = 4
```

Цикл while полезен, когда число повторений заранее неизвестно:

```java
Scanner scanner = new Scanner(System.in);
int number = 0;

while (number != 42) {
    System.out.print("Введите число: ");
    number = scanner.nextInt();
}

System.out.println("Вы угадали!");
```

### Цикл do-while

**Цикл do-while** — цикл с постусловием. Тело цикла выполняется как минимум один раз, затем проверяется условие[3][18][19]:

```java
do {
    // Тело цикла
} while (условие);
```

Пример:

```java
int i = 0;

do {
    System.out.println("i = " + i);
    i++;
} while (i < 5);
// Вывод:
// i = 0
// i = 1
// i = 2
// i = 3
// i = 4
```

Главное отличие от while в том, что тело выполняется хотя бы один раз:

```java
int x = 13;

do {
    System.out.println("x = " + x);
    x--;
} while (x < 10);
// Вывод: x = 13
// (тело выполнилось один раз, хотя условие было ложно)
```

### Цикл foreach (for-each)

**Цикл foreach** — удобный способ перебирать элементы массива или коллекции без явного управления индексом[3]:

```java
for (тип переменная : коллекция) {
    // Код выполняется для каждого элемента
}
```

Пример:

```java
int[] numbers = {1, 2, 3, 4, 5};

for (int num : numbers) {
    System.out.println("Число: " + num);
}
// Вывод:
// Число: 1
// Число: 2
// Число: 3
// Число: 4
// Число: 5

// Со строками
String[] fruits = {"Яблоко", "Банан", "Апельсин"};

for (String fruit : fruits) {
    System.out.println(fruit);
}
// Вывод:
// Яблоко
// Банан
// Апельсин
```

### Вложенные циклы

**Вложенные циклы** — это циклы, находящиеся внутри других циклов. Внутренний цикл выполняется полностью при каждой итерации внешнего цикла[20][21]:

```java
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        System.out.print(i + "," + j + " ");
    }
    System.out.println();
}
```

Вывод:

```
0,0 0,1 0,2
1,0 1,1 1,2
2,0 2,1 2,2
```

Пример: таблица умножения

```java
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 5; j++) {
        System.out.print(i * j + "\t");
    }
    System.out.println();
}
```

Вывод:

```
1	2	3	4	5
2	4	6	8	10
3	6	9	12	15
4	8	12	16	20
5	10	15	20	25
```

### Операторы управления циклом

Для управления потоком выполнения циклов используются специальные операторы:

**break** — прерывает цикл и выходит из него:

```java
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // Выход из цикла
    }
    System.out.println(i);
}
// Вывод: 0, 1, 2, 3, 4
```

**continue** — пропускает оставшуюся часть текущей итерации и переходит к следующей:

```java
for (int i = 0; i < 5; i++) {
    if (i == 2) {
        continue;  // Пропуск текущей итерации
    }
    System.out.println(i);
}
// Вывод: 0, 1, 3, 4
```

## Практические примеры

### Пример 1: Проверка чётности и нечётности

```python
number = int(input("Введите число: "))

if number % 2 == 0:
    print(f"{number} — чётное число")
else:
    print(f"{number} — нечётное число")
```

### Пример 2: Сумма чисел от 1 до N

```cpp
#include <iostream>
using namespace std;

int main() {
    int n = 10;
    int sum = 0;
    
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    
    cout << "Сумма чисел от 1 до " << n << ": " << sum << endl;
    
    return 0;
}
```

### Пример 3: Факториал числа

```javascript
function factorial(n) {
    if (n === 0 || n === 1) {
        return 1;
    }
    
    let result = 1;
    for (let i = 2; i <= n; i++) {
        result *= i;
    }
    
    return result;
}

console.log("Факториал 5: " + factorial(5));  // Вывод: 120
```

### Пример 4: Вложенные циклы — Звёздный треугольник

```python
n = 5

for i in range(1, n + 1):
    for j in range(i):
        print("*", end=" ")
    print()
```

Вывод:

```
*
* *
* * *
* * * *
* * * * *
```

```java
int month = 3;
String season;

switch (month) {

    case 3:
    case 4:
    case 5:
        season = "Весна";
        break;
    case 6:
    case 7:
    case 8:
        season = "Лето";
        break;
    case 9:
    case 10:
    case 11:
        season = "Осень";
        break;
    default:
        season = "Неверный месяц";
}

System.out.println("Сезон: " + season);
// Вывод: Сезон: Весна
```
