# OSINT (Open Source Intelligence)

## Описание
Open Source Intelligence — сбор и анализ информации из открытых источников для разведки, поиска уязвимостей и проведения исследований в области кибербезопасности. OSINT превращает разрозненные куски информации в интересные истории и систематизированные данные.

## Уровень
Средний

## Примеры использования
- Поиск информации о доменах и IP-адресах.
- Сбор данных о целевых организациях.
- Поиск уязвимых устройств и сервисов в интернете.
- Анализ публично доступных данных для пентестинга.

## Ресурсы для изучения
- OSINT методики и инструменты
- Руководства по разведке из открытых источников
- Best practices для этичного OSINT
