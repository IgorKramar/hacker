# База знаний по кибербезопасности, пентестингу и этичному хакерству

Добро пожаловать в базу знаний, основанную на прохождениях машин TryHackMe (Mustacchio, Internal, Lo-Fi) и общих материалах по программированию. Здесь собраны ключевые темы и инструменты для изучения кибербезопасности, пентестинга, этичного хакерства и основ программирования. Каждая тема и инструмент ведёт на отдельную страницу с подробным описанием (в процессе наполнения).

## Инструменты для освоения

| [Gobuster / Dirbuster / ffuf](./tools/gobuster-dirbuster-ffuf.md) | Средний | Разведка: поиск скрытых файлов/путей на веб-сервере. |
| [OpenSSH / ssh](./tools/ssh.md) | Основы | Доступ: вход на сервер после получения ключей. |
| [Find](./tools/find.md) | Средний | Privilege escalation: поиск SUID/SGID бинарников. |
| [Strings / objdump](./tools/strings-objdump.md) | Продвинутый | Анализ: понимание поведения бинарника. |
| [Netcat (nc)](./tools/netcat.md) | Средний | Реверс-шелл: прослушка входящих соединений для получения shell. |
| [Chmod / Chown](./tools/chmod-chown.md) | Основы | Privilege escalation: добавление SUID к bash для рут-доступа. |

| [Excel Viewer / LibreOffice](./tools/excel-libreoffice.md) | Средний | Веб-эксплуатация: тестирование XXE в Excel-документах. |
| [msfvenom](./tools/msfvenom.md) | Продвинутый | Эксплуатация: генерация кода для внедрения в CMS или скрипты. |
| [Metasploit (msfconsole)](./tools/metasploit.md) | Продвинутый | Многоцелевое: настройка reverse shell listeners, брутфорс сервисов, сканирование уязвимостей. |
| [Chisel](./tools/chisel.md) | Продвинутый | Доступ: туннелирование локальных портов. |
| [LinPEAS](./tools/linpeas.md) | Средний | Privilege escalation: автоматический аудит системы на векторы повышения прав. |
| [Python HTTP Server](./tools/python-http-server.md) | Основы | Файловый трансфер: загрузка инструментов на цель; запуск простых веб-серверов. |
| [Rockyou.txt](./tools/rockyou.md) | Основы | Взлом: в комбинации с Burp или Metasploit для атаки на формы. |
| [Python](./tools/python.md) | Основы | Разработка: написание скриптов, обработка исключений (try-except), запуск серверов, автоматизация задач. |
| [Tor](./tools/tor.md) | Средний | Анонимность: обеспечение анонимного доступа в сеть через распределенную сеть узлов, настройка SOCKS5 прокси. |
| [Tornet](./tools/tornet.md) | Средний | Анонимность: автоматическая смена IP-адресов через Tor сеть с заданным интервалом для парсинга и автоматизации. |
| [Shodan](./tools/shodan.md) | Средний | OSINT: поиск уязвимых устройств и сервисов в интернете, сканирование подключенных устройств. |
| [Wi-Fi Map](./tools/wi-fi-map.md) | Средний | OSINT: анализ беспроводных сетей, типов шифрования и географического расположения точек доступа. |
| [Hunter](./tools/hunter.md) | Средний | OSINT: поиск email-адресов, связанных с доменами, верификация подлинности адресов. |
| [Vulners](./tools/vulners.md) | Средний | OSINT: база данных уязвимостей, эксплойтов и патчей, отслеживание развития уязвимостей. |
| [Certy](./tools/certy.md) | Средний | OSINT: база данных SSL и TLS сертификатов, выявление потенциально опасных сертификатов. |
| [Full Hunt](./tools/full-hunt.md) | Средний | OSINT: сканирование веб-приложений на уязвимости, автоматические отчёты и рекомендации. |
| [CodeCrawler](./tools/codecrawler.md) | Средний | OSINT: поиск исходного кода по ключевым словам в публичных репозиториях. |
