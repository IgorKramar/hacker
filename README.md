# База знаний по кибербезопасности, пентестингу и этичному хакерству

Добро пожаловать в базу знаний, основанную на прохождениях машин TryHackMe (Mustacchio, Internal, Lo-Fi) и общих материалах по программированию. Здесь собраны ключевые темы и инструменты для изучения кибербезопасности, пентестинга, этичного хакерства и основ программирования. Каждая тема и инструмент ведёт на отдельную страницу с подробным описанием (в процессе наполнения).
