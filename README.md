# База знаний по кибербезопасности, пентестингу и этичному хакерству

Добро пожаловать в базу знаний, основанную на прохождениях машин TryHackMe (Mustacchio, Internal, Lo-Fi) и общих материалах по программированию. Здесь собраны ключевые темы и инструменты для изучения кибербезопасности, пентестинга, этичного хакерства и основ программирования. Каждая тема и инструмент ведёт на отдельную страницу с подробным описанием (в процессе наполнения).

### Обновления и новые файлы
#### Новые темы (добавлены в `topics/`)
1. **computer-basics.md**: Основы работы компьютера (процессор, биты, данные).
2. **programming-languages.md**: Компилируемые и интерпретируемые языки, машинный код.
3. **ram.md**: Роль оперативной памяти в выполнении программ.
4. **variables-operators.md**: Переменные, условные операторы (if, switch), циклы (for, while).
5. **code-debugging.md**: Отладка, логирование, обработка исключений (try-except).
6. **algorithms-data-structures.md**: Алгоритмы, O-сложность, структуры данных (массивы, списки, деревья, графы).
7. **programming-operations.md**: Арифметические, логические, побитовые операции, функции, классы.
8. **oop.md**: ООП, атрибуты, методы, инкапсуляция, наследование, полиморфизм, абстракция.
9. **solid-principles.md**: Принципы SOLID для проектирования кода.
10. **design-patterns.md**: Порождающие, структурные, поведенческие паттерны.
11. **synchronicity-multithreading-parallelism.md**: Синхронность, многопоточность, параллелизм, race conditions.
12. **websites-servers.md**: Веб-сайты, серверы, процесс загрузки страниц.
13. **frontend-backend.md**: Роли фронтенда и бэкенда в веб-разработке.
14. **http-dns.md**: HTTP-запросы (GET, POST, PUT, DELETE), DNS и кэширование.
15. **databases-basics.md**: Реляционные базы, структура таблиц, ключи.
16. **data-normalization.md**: Нормализация данных, нормальные формы.
17. **nosql-databases.md**: Нереляционные базы (документо-ориентированные, ключ-значение, графовые).
18. **data-caching.md**: Кэширование (браузер, база данных, оперативная память).
19. **deployment-devops.md**: Деплой, автоматизация, DevOps.
20. **ci-cd.md**: Непрерывная интеграция и доставка, настройка задач.
21. **cloud-infrastructure.md**: Облачные сервисы (IaaS, PaaS, SaaS).

3. **docker.md**: Контейнеризация, автоматизация деплоя, изолированные среды.
