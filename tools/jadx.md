# Jadx

## Описание
Jadx — инструмент для декомпиляции APK файлов Android приложений в читаемый Java код с графическим интерфейсом. Позволяет анализировать исходный код приложения, структуру, ресурсы и манифест для выявления уязвимостей безопасности.

## Уровень
Средний

## Примеры использования
- Декомпиляция APK в читаемый Java код.
- Анализ исходного кода приложения для поиска уязвимостей.
- Просмотр структуры приложения и его компонентов.
- Анализ файла манифеста и разрешений.
- Изучение ресурсов и конфигураций приложения.

## Особенности
- Графический интерфейс для удобного анализа.
- Декомпиляция в читаемый Java код.
- Просмотр структуры приложения в виде дерева.
- Поиск по коду и ресурсам.
- Экспорт декомпилированного кода.

## Процесс анализа
- Загрузка APK файла в Jadx.
- Автоматическая декомпиляция в Java код.
- Просмотр исходного кода и структуры.
- Анализ файла манифеста и разрешений.
- Поиск уязвимостей в коде.

## Интеграция с другими инструментами
- Использование вместе с dex2jar для дополнительного анализа.
- Комбинирование с инструментами статического анализа.
- Экспорт кода для дальнейшего анализа.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только собственных приложений или с явного разрешения.
- Понимание юридических последствий незаконного использования.

## Ресурсы для изучения

- Методики статического анализа Android приложений
- Best practices для обратной инженерии
