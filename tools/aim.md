# AIM

## Описание
AIM — инструмент для пассивного поиска информации и нумерации субдоменов (subdomain enumeration) через различные источники данных.

## Уровень
Средний

## Примеры использования
- Пассивный поиск информации о домене: `aim -p passive -d domain.org`
- Нумерация субдоменов для целевого домена.
- Просмотр результатов: `aim names torques`
- Сбор информации о домене из различных источников.

## Ресурсы для изучения
- AIM документация и GitHub
- Методики пассивного поиска субдоменов
- Альтернативные инструменты для enumeration
