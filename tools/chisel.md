# Chisel

## Описание
Инструмент для туннелирования TCP/UDP трафика через HTTP, используется для проброса портов и обхода файрволов.

## Уровень
Продвинутый

## Примеры использования
- Запуск сервера: `chisel server --port 8000 --reverse`
- Подключение клиента: `chisel client server.com:8000 R:8080:localhost:80`
- Туннелирование локальных портов.

## Ресурсы для изучения
- [Chisel GitHub](https://github.com/jpillora/chisel)
- Port forwarding методики
