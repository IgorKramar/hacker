# Wi-Fi Pineapple

## Описание
Wi-Fi Pineapple Enterprise — профессиональное устройство для проведения беспроводных атак: автоматическое создание Evil Twin сетей, сканирование сетей, создание сильного сигнала для автоматического подключения устройств, подмена DNS.

## Уровень
Продвинутый

## Примеры использования

### Автоматическое создание Evil Twin
- Устройство сканирует сети и создает сильный сигнал.
- Автоматическое подключение устройств к поддельным сетям.
- Пример: телефон, подключенный к сети на конференции, автоматически подключается к сети в кофейне.
- Wi-Fi Pineapple перехватывает эти сигналы и создает "злого близнеца".

### Дополнительные возможности
- Подмена DNS для перенаправления запросов на поддельные сайты.
- Контроль браузера жертвы через подмену DNS.
- Выполнение различных атак через контролируемый трафик.

### Профессиональное использование
- Более заметное устройство по сравнению с Flipper Zero.
- Используется профессиональными хакерами для сложных атак.
- Подготовка к беспроводным атакам в общественных местах.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных сетях или с явного разрешения.
- Понимание юридических последствий незаконного использования.
