# Medusa Fisher

## Описание
Medusa Fisher — инструмент для создания фишинговых сайтов с QR-кодами, позволяющий создавать поддельные страницы входа в систему (например, Instagram) и перехватывать учётные данные жертв.

## Уровень
Продвинутый

## Примеры использования
- Создание фишинговых сайтов для различных сервисов (Instagram и другие).
- Выбор шаблона для фишингового сайта.
- Настройка сервиса туннелирования на localhost.
- Размещение фишинговой страницы через интернет.
- Генерация QR-кодов с вредоносными ссылками.
- Сохранение учётных данных жертвы в реальном времени.
- Создание пользовательских страниц входа в систему.

## Установка
- Клонирование репозитория Medusa Fisher с GitHub.
- Запуск установочного скрипта `setup.sh`.
- Генерация SSH-ключей для туннелирования.
