# Nmap

## Описание
Nmap (Network Mapper) — инструмент для сканирования сетей, обнаружения открытых портов и сервисов на целевых системах.

## Уровень
Основы

## Примеры использования
- Базовое сканирование портов: `nmap -sC -sV <target>`
- Обнаружение активных хостов: `nmap -sn <network>`
- Определение версий сервисов: `nmap -sV <target>`
