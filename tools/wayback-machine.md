# Wayback Machine

## Описание
Wayback Machine — веб-архив для поиска старых версий сайтов, обнаружения удаленного контента, административных панелей и чувствительной информации.

## Уровень
Средний

## Примеры использования
- Поиск старых версий сайта через веб-интерфейс.
- Фильтрация страниц, связанных с админкой.
- Обнаружение удаленных файлов и контента.
- Получение исторических данных о сайте.

## Ресурсы для изучения
- [Wayback Machine](https://web.archive.org/)
- Инструменты для автоматизации (waybackurls)
- Методики поиска в веб-архивах
