# Hping3

## Описание
Hping3 — инструмент для отправки кастомных TCP/IP пакетов: скрытые ICMP-пинги, сканирование портов, тестирование файрволов и сетевой анализ.

## Уровень
Средний

## Примеры использования
- Скрытые ICMP-пинги: `hping3 -1 target.com`
- Сканирование портов: `hping3 -S -p 80 target.com`
- Тестирование файрволов и правил фильтрации.
- Отправка кастомных пакетов для сетевого анализа.

## Ресурсы для изучения
- Hping3 документация
- Методики скрытого сканирования
- Альтернативные инструменты (nmap, masscan)
