# Tornet

## Описание
Python библиотека для автоматической смены IP-адресов через Tor сеть с заданным интервалом, обеспечивающая анонимность и ротацию IP для автоматизированных процессов.

## Уровень
Средний

## Примеры использования
- Установка: `pip install tornet` (может потребоваться `--break-system-packages` на некоторых системах)
- Запуск с интервалом смены IP: `tornet --interval 5`
- Установка неограниченного количества смен IP.
- Интеграция в Python скрипты для автоматизации.

## Ресурсы для изучения
- Tornet GitHub репозиторий
- Python библиотеки для работы с Tor
- Документация по автоматизации смены IP
