# Mimikatz

## Описание
Mimikatz — утилита для извлечения учетных данных из систем Windows, обходя аутентификацию через NTLM-хэши и тикеты Kerberos. Позволяет извлекать текстовые пароли и хэши NTLM из процесса LSASS и создавать "золотые билеты" для неограниченного доступа к службам.

## Уровень
Продвинутый

## Примеры использования

### Извлечение учетных данных
- Сбор учетных данных из систем Windows.
- Обход аутентификации через NTLM-хэши и тикеты Kerberos.
- Извлечение текстовых паролей и хэшей NTLM из процесса LSASS.

### Создание золотых билетов
- Создание "золотого билета" для неограниченного доступа к службам через Kerberos.
- Использование тикетов Kerberos для обхода аутентификации.
- Получение доступа к службам без знания паролей.

### Дополнительные возможности
- Извлечение хэшей паролей из памяти.
- Работа с тикетами Kerberos.
- Обход защиты Windows для извлечения учетных данных.

## Защита от Mimikatz
- Ограничение доступа к процессу LSASS.
- Использование защищенных учетных данных Windows.
- Мониторинг подозрительной активности, связанной с извлечением учетных данных.
- Регулярное обновление систем для защиты от известных уязвимостей.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных системах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
