# Wireshark

## Описание
Wireshark — мощный инструмент для анализа сетевого трафика в реальном времени, позволяющий захватывать и анализировать пакеты данных, проходящие через сетевой интерфейс. Используется для отладки сетевых приложений, анализа безопасности и расследования сетевых инцидентов.

## Уровень
Средний

## Примеры использования

### Анализ сетевого трафика
- Захват сетевого трафика в реальном времени.
- Фильтрация трафика по различным критериям (IP-адрес, порт, протокол).
- Анализ пакетов данных для понимания сетевой активности.

### Отслеживание IP-адресов
- Фильтрация трафика по локальному IP-адресу.
- Получение публичных IP-адресов из сетевого трафика.
- Определение местоположения по IP-адресам.

### Анализ протоколов
- Анализ различных сетевых протоколов (HTTP, HTTPS, DNS, TCP, UDP).
- Расшифровка зашифрованного трафика (при наличии ключей).
- Поиск уязвимостей в сетевых приложениях.
- Наблюдение за трафиком данных в фемтосотах и IoT устройствах.
- Анализ мобильного трафика для декодирования SMS и голосовых вызовов (с подходящими кодеками).

### Расследование инцидентов
- Анализ сетевых атак и подозрительной активности.
- Восстановление переданных данных из пакетов.
- Документирование сетевых событий для отчетов.

## Установка
- Скачивание Wireshark с официального сайта для Windows, Linux и macOS.
- Установка необходимых драйверов для захвата пакетов (WinPcap на Windows).

## Фильтры
- Фильтрация по IP-адресу: `ip.addr == 192.168.1.1`
- Фильтрация по порту: `tcp.port == 80`
- Фильтрация по протоколу: `http`, `dns`, `tcp`
- Комбинирование фильтров для точного поиска.

## Безопасность
- Использование только в легальных целях и с разрешения.
- Защита конфиденциальных данных при анализе трафика.
- Понимание юридических последствий перехвата трафика.
