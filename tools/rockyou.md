# Rockyou.txt

## Описание
Популярный словарь паролей (rockyou.txt) содержащий миллионы реальных паролей, используется для брутфорса и взлома паролей.

## Уровень
Основы

## Примеры использования
- Использование с Burp Suite Intruder для брутфорса форм.
- Использование с Metasploit для брутфорса сервисов.
- Использование с Hashcat/John для взлома хэшей.

## Ресурсы для изучения
- Расположение: `/usr/share/wordlists/rockyou.txt`
- Дополнительные wordlists: SecLists
