# Stool

## Описание
Stool — инструмент для социальной инженерии: создание фейковых сайтов, логинов, QR-кодов, проведение фишинговых атак и массовая рассылка писем.

## Уровень
Продвинутый

## Примеры использования
- Создание QR-кода с ссылкой на сайт.
- Создание фейковых сайтов для фишинговых атак.
- Создание фейковых форм логина.
- Отправка массовых писем.
- Проведение атак социальной инженерии.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных системах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
