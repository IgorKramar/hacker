# Tor

## Описание
Tor (The Onion Router) — программное обеспечение и сеть для анонимной коммуникации, обеспечивающая сокрытие IP-адреса и местоположения пользователя через многослойное шифрование и маршрутизацию.

## Уровень
Средний

## Примеры использования
- Установка Tor: `sudo apt install tor` (Linux) или скачать с torproject.org
- Запуск Tor сервиса: `sudo systemctl start tor`
- Использование как SOCKS5 прокси: `localhost:9050`
- Настройка браузера для работы через Tor прокси.
