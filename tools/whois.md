# Whois

## Описание
Whois — инструмент для получения информации о домене: владелец, контактные данные, серверы имен, дата регистрации и срок действия домена.

## Уровень
Средний

## Примеры использования
- Получение информации о домене: `whois example.com`
- Получение контактных данных владельца.
- Определение серверов имен (nameservers).
- Получение даты регистрации и срока действия домена.

## Ресурсы для изучения
- Whois документация
- Методики OSINT через whois
- Онлайн whois сервисы
