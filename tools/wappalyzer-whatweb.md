# Wappalyzer / WhatWeb

## Описание
Инструменты для веб-анализа: определение используемых технологий на сайте (CMS, фреймворки, серверное ПО, библиотеки) для выявления потенциальных уязвимостей.

## Уровень
Средний

## Примеры использования
- Определение CMS: `whatweb http://target.com`
- Выявление используемых фреймворков и библиотек.
- Определение версий серверного ПО.
- Использование Wappalyzer (браузерное расширение) для быстрого анализа.

## Ресурсы для изучения
- [WhatWeb GitHub](https://github.com/urbanadventurer/WhatWeb)
- [Wappalyzer](https://www.wappalyzer.com/)
- Методики веб-анализа технологий
