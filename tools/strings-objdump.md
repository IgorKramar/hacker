# Strings / objdump

## Описание
Инструменты для анализа бинарных файлов: strings для извлечения текстовых строк, objdump для дизассемблирования и понимания поведения бинарника.

## Уровень
Продвинутый

## Примеры использования
- Извлечение строк: `strings binary_file`
- Поиск конкретных строк: `strings binary_file | grep "password"`
- Дизассемблирование: `objdump -d binary_file`
- Анализ секций: `objdump -h binary_file`

## Ресурсы для изучения
- Linux binary analysis инструменты
- Материалы по reverse engineering
