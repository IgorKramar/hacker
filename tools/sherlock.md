# Sherlock

## Описание
Sherlock — инструмент для поиска уязвимостей и перечисления в Windows системах. Позволяет автоматически находить известные уязвимости и векторы эскалации привилегий.

## Уровень
Средний

## Примеры использования
- Поиск уязвимостей в Windows системах.
- Перечисление и проверка системы на наличие известных уязвимостей.
- Автоматическое обнаружение векторов эскалации привилегий.
- Анализ конфигурации системы.

## Особенности
- Автоматический поиск уязвимостей.
- Перечисление системы для поиска слабых мест.
- Простота использования и эффективность.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных системах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
