# Sherlock

## Описание
Sherlock — инструмент для проверки наличия пользовательских имён (username) в различных социальных сетях и веб-сервисах, помогает находить аккаунты по никнейму.

## Уровень
Средний

## Примеры использования
- Поиск пользователя по никнейму: `sherlock network david`
- Проверка наличия аккаунтов с указанным именем пользователя.
- Поиск ссылок на сайты с пользовательским именем.
- Сбор информации о присутствии пользователя в различных сервисах.

## Ресурсы для изучения
