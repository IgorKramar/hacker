# Angry IP Scanner

## Описание
Angry IP Scanner — кроссплатформенный инструмент для быстрого сканирования IP-адресов и портов в локальных и общедоступных сетях. Позволяет обнаруживать активные устройства, открытые порты и получать доступ к веб-интерфейсам устройств.

## Уровень
Средний

## Примеры использования

### Сканирование локальной сети
- Автоматическое определение диапазона IP-адресов сети.
- Сканирование устройств в локальной сети (телефоны, ноутбуки, смарт-телевизоры).
- Определение активных устройств по цветовой индикации:
  - Зелёный — устройство реагирует на пинг.
  - Красный — устройство не реагирует.
  - Синий — устройство подключено, но без открытых портов.

### Работа с портами
- Сканирование популярных портов: 21 (FTP), 22 (SSH), 23 (Telnet), 80 (HTTP), 445 (SMB), 3389 (RDP), 5900 (VNC).
- Отображение только устройств с открытыми портами.
- Открытие IP-адресов в веб-браузере для доступа к страницам входа в систему маршрутизатора.

### Сканирование общедоступных IP-адресов
- Сканирование общедоступных IP-адресов из интернета.
- Использование сайтов для выбора диапазона IP-адресов (например, nirsoft.net/countryip).
- Анализ результатов сканирования для обнаружения открытых портов.

## Установка
- Скачивание программы с сайта angryip.org для Windows, Linux и других платформ.
- При установке рекомендуется снять флажок "Запустить злой IP-сканер" для безопасности.

## Безопасность
- Открытые порты SSH, Telnet или FTP представляют риск для безопасности.
- Рекомендуется использовать VPN для сокрытия следов при сканировании.
- Для предотвращения сканирования IP-адресов рекомендуется закрыть все порты на маршрутизаторе.
