# TCPdump

## Описание
TCPdump — инструмент командной строки для захвата и анализа сетевого трафика, сохранения пакетов для дальнейшего анализа.

## Уровень
Средний

## Примеры использования
- Захват трафика: `tcpdump -i eth0`
- Сохранение в файл: `tcpdump -w capture.pcap`
- Фильтрация по протоколу: `tcpdump tcp port 80`
- Отслеживание DNS-запросов: `tcpdump port 53`
- Наблюдение за трафиком данных в фемтосотах и IoT устройствах
- Перехват мобильного трафика для анализа

## Ресурсы для изучения
- TCPdump документация
- Методики анализа сетевого трафика
- Альтернативные инструменты (tshark, Wireshark)
