# SSLScan

## Описание
SSLScan — инструмент для сканирования SSL/TLS сертификатов, проверки их версии, алгоритмов шифрования и выявления потенциальных уязвимостей в конфигурации SSL/TLS.

## Уровень
Средний

## Примеры использования
- Сканирование SSL-сертификата: `sslscan example.com`
- Получение информации об IP-адресе, версии сертификата и его владельце.
- Проверка поддерживаемых протоколов и алгоритмов шифрования.
- Выявление устаревших или небезопасных конфигураций SSL/TLS.

## Ресурсы для изучения
- SSLScan документация
- Методики анализа SSL/TLS сертификатов
- Best practices для безопасной конфигурации SSL/TLS
