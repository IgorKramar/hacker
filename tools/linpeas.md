# LinPEAS (Linux Privilege Escalation Awesome Script)

## Описание
LinPEAS — скрипт для повышения привилегий в Linux, аналогичный версии для Windows. Выполняет цветовую координацию и выявляет уязвимости. Простота установки и отсутствие зависимостей делают его популярным инструментом для автоматизированного перечисления.

## Уровень
Средний

## Примеры использования

### Автоматизированное перечисление
- Выполняет автоматическое перечисление и выявляет уязвимости.
- Анализ процессов, cron, сервисов.
- Сбор сетевой информации, DNS, IP-адресов и активных портов.
- Проверка информации о пользователях и входах в систему.

### Анализ результатов
- Оценка вероятности привилегированного вектора эскалации по цветам.
- Получение информации о версии Linux и доступных папках.
- Автоматическое выполнение команд и быстрое получение результатов.

### Информация о пользователях
- Идентификация суперпользователей, включая root.
- Проверка последних входов в систему и источников входа.
- Хранение информации о пользователях и группах.

### Анализ уязвимостей системы
- Проверка политики использования паролей и информации о программном обеспечении.
- Обнаружение SSH, разрешающего вход с правами root.
- Проверка изменений файлов за последние минуты для выявления возможных уязвимостей.

### Поиск паролей
- Программа ищет возможные пароли в истории пакетов.
- Нахождение потенциального пароля и других интересных данных.
- Ограничение поиска паролей до 70 для избежания перегрузки информацией.

### Визуальная оценка
- Визуальная оценка информации с помощью красно-жёлтых цветов.
- Повторный просмотр для выявления дополнительных уязвимостей.
- Пример обнаружения пароля root в истории.

## Особенности
- Цветовая координация для быстрой оценки уязвимостей.
- Простота установки и отсутствие зависимостей.
- Автоматическое выполнение команд для перечисления.
- Быстрое получение результатов.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных системах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
