# LinPEAS

## Описание
Linux Privilege Escalation Awesome Script — автоматизированный скрипт для аудита Linux систем на векторы повышения привилегий.

## Уровень
Средний

## Примеры использования
- Загрузка на целевую систему.
- Запуск: `./linpeas.sh`
- Анализ результатов для поиска векторов privilege escalation.

## Ресурсы для изучения
- [LinPEAS GitHub](https://github.com/carlospolop/PEASS-ng)
- Методики автоматизированного enumeration
