# Find

## Описание
Linux команда для поиска файлов и директорий, часто используется для поиска SUID/SGID бинарников при privilege escalation.

## Уровень
Средний

## Примеры использования
- Поиск SUID файлов: `find / -perm -4000 2>/dev/null`
- Поиск SGID файлов: `find / -perm -2000 2>/dev/null`
- Поиск файлов по имени: `find / -name "*.txt" 2>/dev/null`
