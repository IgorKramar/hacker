# Perpsuit Community Edition

## Описание
Perpsuit Community Edition — инструмент для перехвата и анализа HTTP-запросов, позволяющий модифицировать запросы для тестирования безопасности веб-приложений. Используется для анализа форм, API и обнаружения уязвимостей.

## Уровень
Средний

## Примеры использования
- Перехват HTTP-запросов для анализа.
- Анализ формы для входа в систему и отправка запроса на API.
- Модификация запросов для тестирования уязвимостей.
- Добавление полей в запросы для тестирования Mass Assignment.
- Анализ структуры API и поиск скрытых функций.

## Особенности
- Перехват и модификация HTTP-запросов.
- Анализ структуры запросов и ответов.
- Тестирование уязвимостей веб-приложений.
- Community Edition с базовым функционалом.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных приложениях или с явного разрешения.
- Понимание юридических последствий незаконного использования.

- Методики тестирования безопасности веб-приложений
- Best practices для работы с HTTP-запросами
