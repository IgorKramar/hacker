# DnSpy

## Описание
DnSpy — отладчик и редактор .NET-программ, позволяющий анализировать, редактировать и отлаживать приложения на .NET Framework и .NET Core без исходного кода.

## Уровень
Продвинутый

## Примеры использования

### Анализ .NET-программ
- Добавление файлов в отладчик для анализа.
- Обзор интерфейса отладчика: список бинарных файлов, информация о сборке.
- Определение точки входа в программу.
- Поиск механизмов проверки авторизации в пространствах имён.
- Анализ методов, отвечающих за попытку логина.

### Отладка программ
- Настройка брейкпоинтов для остановки выполнения программы.
- Пошаговое выполнение программы и анализ её состояния.
- Изучение переменных и параметров во время выполнения.

### Редактирование программ
- Редактирование IL-кода для изменения поведения программы.
- Обход защитных механизмов и авторизации.
- Модификация логики программы без исходного кода.

## Особенности
- Поддержка .NET Framework и .NET Core.
- Интегрированный отладчик и редактор.
- Декомпиляция IL-кода в C#.
- Работа с реестром Windows для обхода авторизации.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных программах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
