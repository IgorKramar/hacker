# Python HTTP Server

## Описание
Встроенный HTTP-сервер Python для быстрого развертывания локального веб-сервера, используется для файлового трансфера и тестирования.

## Уровень
Основы

## Примеры использования
- Запуск сервера: `python3 -m http.server 8000`
- Загрузка инструментов на цель через wget/curl.
- Тестирование веб-приложений локально.

## Ресурсы для изучения
- Python http.server документация
- Файловый трансфер методики
