# DNSN

## Описание
DNSN — инструмент для поиска информации и нумерации DNS-серверов, получения данных о субдоменах и DNS-записях целевого домена.

## Уровень
Средний

## Примеры использования
- Поиск информации о DNS: `dnsn hack.site.org`
- Нумерация DNS-серверов для целевого домена.
- Получение информации о субдоменах.
- Сбор DNS-данных для анализа инфраструктуры.

## Ресурсы для изучения
- DNSN документация
- Методики DNS enumeration
- Альтернативные инструменты для анализа DNS
