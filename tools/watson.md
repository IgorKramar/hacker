# Watson

## Описание
Watson — инструмент для проверки уязвимостей в Windows системах. Позволяет автоматически находить известные уязвимости в системе на основе версии Windows и установленных обновлений.

## Уровень
Средний

## Примеры использования
- Автоматическая проверка системы на наличие известных уязвимостей.
- Сопоставление версии Windows с базой данных уязвимостей.
- Поиск векторов эскалации привилегий.
- Анализ установленных обновлений и патчей.

## Особенности
- Автоматическое обнаружение уязвимостей.
- Сопоставление с базой данных известных уязвимостей.
- Простота использования и эффективность.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных системах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
