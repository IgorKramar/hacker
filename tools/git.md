# Git

## Описание
Система контроля версий для управления кодом, работы с репозиториями, интеграции с CI/CD системами и отслеживания изменений.

## Уровень
Основы

## Примеры использования
- Клонирование репозитория: `git clone <repo>`
- Коммит изменений: `git commit -m "message"`
- Работа с ветками: `git branch`, `git checkout`
- Интеграция с CI/CD пайплайнами.

## Ресурсы для изучения
- [Git Documentation](https://git-scm.com/doc)
- Git для DevOps материалы
