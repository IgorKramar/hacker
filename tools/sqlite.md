# SQLite Browser / sqlite3

## Описание
Инструменты для работы с SQLite базами данных: SQLite Browser (GUI) и sqlite3 (CLI) для анализа .db файлов и извлечения данных.

## Уровень
Средний

## Примеры использования
- Открытие .db файла: `sqlite3 database.db`
- Просмотр таблиц: `.tables`
- Извлечение хэшей паролей: `SELECT * FROM users;`
- Чтение баз данных Android приложений для динамического анализа.
- Анализ хранения конфиденциальных данных в мобильных приложениях.

## Ресурсы для изучения
- [SQLite Documentation](https://www.sqlite.org/docs.html)
- [DB Browser for SQLite](https://sqlitebrowser.org/)
