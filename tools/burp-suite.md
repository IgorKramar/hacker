# Burp Suite

## Описание
Комплексный инструмент для тестирования безопасности веб-приложений: перехват и модификация HTTP-запросов, поиск уязвимостей, брутфорс логинов.

## Уровень
Продвинутый

## Примеры использования
- Настройка Proxy для перехвата запросов.
- Использование Intruder для брутфорса форм.
- Анализ LFI уязвимостей через Repeater.
- Сканирование на уязвимости с помощью Scanner.
