# Wi-Fi Map

## Описание
Wi-Fi Map — инструмент для сбора и анализа данных о беспроводных сетях по всему миру, включая типы шифрования и географическое расположение точек доступа.

## Уровень
Средний

## Примеры использования
- Анализ типов шифрования Wi-Fi сетей (WEP, WPA, WPA2, WPA3).
- Выявление сетей с устаревшими методами защиты.
- Географический анализ расположения точек доступа.
- Исследование распространения уязвимых беспроводных сетей.

## Ресурсы для изучения
- Wi-Fi Map документация
- Методики анализа беспроводных сетей
