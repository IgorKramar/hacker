# Stormbreaker

## Описание
Stormbreaker — инструмент для отслеживания устройств и получения доступа к камере, микрофону и местоположению через веб-браузер. Предоставляет веб-панель администратора для управления отслеживаемыми устройствами и отправки ссылок на поддельные веб-сайты.

## Уровень
Продвинутый

## Примеры использования
- Отслеживание устройств через веб-браузер.
- Получение доступа к камере, микрофону и местоположению устройств.
- Управление отслеживаемыми устройствами через панель администратора.
- Отправка ссылок на веб-сайты для доступа к устройствам.
- Использование шаблонов веб-сайтов (камера, микрофон, "рядом с вами").
- Сбор данных об операционной системе и браузере.

## Установка
- Клонирование репозитория с GitHub: `git clone <repository_url>`.
- Обновление системы и установка необходимых пакетов: `python3-requests`, `python3-colorama`, `python3-psutil`.
- Запуск установочного скрипта: `bash install.sh`.
- Получение ссылки на панель администратора.

## Интеграция с ngrok
- Использование ngrok для доступа к Stormbreaker за пределами локальной сети.
- Настройка туннелирования для доступа к панели администратора из интернета.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных устройствах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
- Несанкционированный доступ к устройствам других людей является незаконным.
