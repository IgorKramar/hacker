# Skipfish

## Описание
Skipfish — автоматический инструмент для сканирования веб-приложений, позволяющий охватывать все возможные слова в API для обнаружения скрытых функций и эндпоинтов. Используется для поиска недокументированных API и уязвимостей.

## Уровень
Средний

## Примеры использования
- Охват всех возможных слов в API для поиска скрытых функций.
- Обнаружение недокументированных эндпоинтов.
- Анализ документации API и поиск скрытых функций.
- Автоматическое сканирование веб-приложений на уязвимости.
- Поиск скрытой функциональности через перебор возможных путей.

## Особенности
- Автоматическое сканирование веб-приложений.
- Поиск скрытых эндпоинтов и функций.
- Анализ структуры API.
- Обнаружение недокументированных функций.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных приложениях или с явного разрешения.
- Понимание юридических последствий незаконного использования.

## Ресурсы для изучения

- Методики поиска скрытых функций API
- Best practices для тестирования безопасности веб-приложений
