# Intelligence X

## Описание
Intelligence X — платформа для поиска информации в утечках данных, позволяющая проверять email-адреса, номера телефонов и другие данные на наличие в различных базах данных утечек. Используется для OSINT исследований и проверки безопасности данных.

## Уровень
Средний

## Примеры использования
- Проверка адреса электронной почты на наличие в утечках данных.
- Поиск утечек данных, содержащих физические адреса.
- Загрузка информации об утечке данных для получения дополнительной информации.
- Поиск информации о людях в различных базах данных утечек.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных данных или с явного разрешения.
- Понимание юридических последствий незаконного использования.
- Соблюдение законов о защите персональных данных.
