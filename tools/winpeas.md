# WinPEAS (Windows Privilege Escalation Awesome Script)

## Описание
WinPEAS — скрипт для повышения привилегий в Windows, аналогичный версии для Linux (LinPEAS). Выполняет цветовую координацию и выявляет уязвимости. Простота установки и отсутствие зависимостей делают его популярным инструментом для автоматизированного перечисления Windows систем.

## Уровень
Средний

## Примеры использования

### Автоматизированное перечисление
- Выполняет автоматическое перечисление и выявляет уязвимости.
- Анализ процессов, сервисов, сетевой информации.
- Проверка информации о пользователях и входах в систему.
- Поиск паролей и конфиденциальных данных.

### Анализ результатов
- Оценка вероятности привилегированного вектора эскалации по цветам.
- Получение информации о версии Windows и доступных папках.
- Автоматическое выполнение команд и быстрое получение результатов.

### Информация о системе
- Идентификация версии Windows и архитектуры.
- Проверка установленных программ и сервисов.
- Анализ конфигурации системы на предмет слабых мест.

### Поиск уязвимостей
- Поиск известных уязвимостей в системе.
- Проверка конфигурации на предмет неправильных настроек.
- Выявление векторов эскалации привилегий.

## Особенности
- Цветовая координация для быстрой оценки уязвимостей.
- Простота установки и отсутствие зависимостей.
- Автоматическое выполнение команд для перечисления.
- Быстрое получение результатов.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только на собственных системах или с явного разрешения.
- Понимание юридических последствий незаконного использования.
