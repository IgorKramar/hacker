# ADB (Android Debug Bridge)

## Описание
ADB (Android Debug Bridge) — инструмент командной строки для управления устройствами Android и эмуляторами. Позволяет устанавливать приложения, извлекать данные, получать доступ к файловой системе и выполнять различные операции для тестирования безопасности Android приложений.

## Уровень
Средний

## Примеры использования

### Управление устройствами
- Просмотр подключенных устройств: `adb devices`
- Установка приложений: `adb install <apk_file>`
- Удаление приложений: `adb uninstall <package_name>`
- Получение информации об устройстве.

### Доступ к данным приложений
- Извлечение APK файлов: `adb pull <path>`
- Установка APK файлов: `adb install <apk_file>`
- Доступ к каталогам приложений через файловую систему.
- Просмотр баз данных приложений.
- Чтение общих настроек (SharedPreferences).

### Динамический анализ
- Доступ к логам приложения: `adb logcat`
- Мониторинг активности приложения.
- Извлечение данных для анализа.
- Установка и тестирование приложений в эмуляторах.

### Работа с файловой системой
- Просмотр файлов: `adb shell ls`
- Навигация по директориям: `adb shell cd`
- Копирование файлов: `adb pull` и `adb push`
- Выполнение команд на устройстве: `adb shell`

## Интеграция с другими инструментами
- Использование вместе с Burp Suite для перехвата трафика.
- Работа с SQLite для анализа баз данных.
- Комбинирование с инструментами статического анализа.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только собственных устройств или с явного разрешения.
- Понимание юридических последствий незаконного использования.
