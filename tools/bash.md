# Bash

## Описание
Bash shell для создания скриптов автоматизации, кастомных инструментов для path hijacking и выполнения команд в Linux системах.

## Уровень

- Path hijacking через кастомные скрипты.
- Выполнение команд и обработка вывода.

- Linux shell scripting материалы
