# ngrok

## Описание
ngrok — инструмент для создания безопасных туннелей к локальным серверам, позволяющий предоставлять доступ к локальным приложениям из интернета. Используется для тестирования веб-приложений, демонстрации проектов и доступа к локальным сервисам извне.

## Уровень
Средний

## Примеры использования
- Создание туннелей к локальным серверам для доступа из интернета.
- Предоставление доступа к локальным приложениям без настройки портов роутера.
- Тестирование веб-приложений на реальных устройствах.
- Демонстрация проектов клиентам или коллегам.
- Интеграция с инструментами пентестинга для доступа к панелям управления из интернета.

## Установка
- Загрузка ngrok с официального сайта.
- Регистрация в сервисе ngrok: ввод имени, электронной почты и пароля.
- Подключение учётной записи к операционной системе с помощью токена аутентификации.

## Настройка
- Создание туннеля к локальному порту: `ngrok http <port>`.
- Получение публичной ссылки для доступа к локальному серверу.
- Настройка доменного имени для постоянного доступа (опционально).

## Безопасность
- Использование HTTPS для безопасного туннелирования.
- Ограничение доступа через настройки ngrok.
- Регулярное обновление токена аутентификации.
