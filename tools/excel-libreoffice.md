# Excel Viewer / LibreOffice

## Описание
Инструменты для работы с Excel-документами, используемые для тестирования XXE уязвимостей в офисных документах.

## Уровень
Средний

## Примеры использования
- Открытие Excel-файлов с XXE payload.
- Анализ структуры XML в .xlsx файлах.
- Тестирование обработки внешних сущностей.

## Ресурсы для изучения
- XXE уязвимости документация
- Методики тестирования XXE в офисных документах
