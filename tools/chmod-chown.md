# Chmod / Chown

## Описание
Linux команды для управления правами доступа к файлам: chmod для изменения прав, chown для изменения владельца.

## Уровень
Основы

## Примеры использования
- Добавление SUID: `chmod +s /usr/bin/bash`
- Изменение прав: `chmod 755 script.sh`
- Изменение владельца: `chown root:root file.txt`
