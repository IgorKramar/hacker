# dex2jar

## Описание
dex2jar — инструмент для преобразования DEX файлов (Dalvik Executable) Android приложений в JAR файлы, что позволяет анализировать код приложения с помощью стандартных инструментов для Java. Используется для статического анализа Android приложений и обратной инженерии.

## Уровень
Средний

## Примеры использования
- Преобразование DEX файлов в JAR для анализа кода.
- Декомпиляция Android приложений для статического анализа.
- Извлечение исходного кода из скомпилированных APK.
- Анализ структуры и логики приложения.

## Процесс работы
- Извлечение DEX файлов из APK.
- Преобразование DEX в JAR формат.
- Использование Java декомпиляторов для анализа кода.
- Изучение логики и структуры приложения.

## Интеграция с другими инструментами
- Использование вместе с Java декомпиляторами (JD-GUI, Fernflower).
- Анализ полученных JAR файлов стандартными инструментами.
- Комбинирование с Jadx для более полного анализа.

## Важные предупреждения
- Использование только в образовательных целях и с разрешения.
- Тестирование только собственных приложений или с явного разрешения.
- Понимание юридических последствий незаконного использования.
