# Netdiscover

## Описание
Netdiscover — инструмент для активного/пассивного сканирования сети, обнаружения активных хостов и их IP-адресов в локальной сети.

## Уровень
Средний

## Примеры использования
- Сканирование локальной сети для обнаружения активных хостов: `netdiscover -r 192.168.1.0/24`
- Определение IP-адресов устройств в сети.
- Поиск целевых машин для пентестинга.
- Пассивное сканирование сети без отправки пакетов.

## Ресурсы для изучения
- Netdiscover документация
- Методики сканирования сетей
- Альтернативные инструменты (arp-scan, nmap)
