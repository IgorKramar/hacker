# Dig

## Описание
Dig (Domain Information Groper) — инструмент для изучения DNS: получение IP-адресов, поддоменов, MX-записей, TXT-записей и других DNS-данных.

## Уровень
Средний

## Примеры использования
- Получение A-записей: `dig example.com`
- Получение MX-записей: `dig MX example.com`
- Получение TXT-записей: `dig TXT example.com`
- Получение всех записей: `dig ANY example.com`

## Ресурсы для изучения
- Dig документация
- DNS enumeration методики
- Альтернативные DNS инструменты (nslookup, host)
