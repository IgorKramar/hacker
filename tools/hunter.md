# Hunter

## Описание
Hunter — инструмент для поиска email-адресов, связанных с доменами, с предоставлением дополнительной информации о подлинности адресов.

## Уровень
Средний

## Примеры использования
- Поиск email-адресов сотрудников организации по домену.
- Верификация подлинности email-адресов.
- Сбор контактной информации для OSINT.
- Использование в этичных пентестах и маркетинге.

## Ресурсы для изучения
- [Hunter.io](https://hunter.io/)
- Email enumeration методики
- Best practices для защиты от email enumeration
