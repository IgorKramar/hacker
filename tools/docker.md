# Docker

## Описание
Платформа для контейнеризации приложений, автоматизации деплоя, тестирования в изолированных средах и управления контейнерами.

## Уровень
Средний

## Примеры использования
- Создание контейнера: `docker run -it image_name`
- Сборка образа: `docker build -t image_name .`
- Управление контейнерами: `docker ps`, `docker exec`
- Автоматизация деплоя через Docker Compose.

## Ресурсы для изучения
- [Docker Documentation](https://docs.docker.com/)
- Docker security и best practices
