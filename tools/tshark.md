# Tshark

## Описание
Tshark — CLI версия Wireshark для захвата и анализа сетевого трафика, извлечения конкретных данных из пакетов и анализа протоколов.

## Уровень
Средний

## Примеры использования
- Захват трафика: `tshark -i eth0`
- Извлечение DNS-запросов: `tshark -r capture.pcap -Y "dns"`
- Фильтрация по протоколу: `tshark -Y "http"`
- Сохранение в файл: `tshark -w output.pcap`
