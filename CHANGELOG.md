# Changelog

История изменений базы знаний по кибербезопасности, пентестингу и этичному хакерству.

---

## Обновления и новые файлы

### Новые темы (добавлены в `topics/`)

1. **computer-basics.md**: Основы работы компьютера (процессор, биты, данные).
2. **programming-languages.md**: Компилируемые и интерпретируемые языки, машинный код.
3. **ram.md**: Роль оперативной памяти в выполнении программ.
4. **variables-operators.md**: Переменные, условные операторы (if, switch), циклы (for, while).
5. **code-debugging.md**: Отладка, логирование, обработка исключений (try-except).
6. **algorithms-data-structures.md**: Алгоритмы, O-сложность, структуры данных (массивы, списки, деревья, графы).
7. **programming-operations.md**: Арифметические, логические, побитовые операции, функции, классы.
8. **oop.md**: ООП, атрибуты, методы, инкапсуляция, наследование, полиморфизм, абстракция.
9. **solid-principles.md**: Принципы SOLID для проектирования кода.
10. **design-patterns.md**: Порождающие, структурные, поведенческие паттерны.
11. **synchronicity-multithreading-parallelism.md**: Синхронность, многопоточность, параллелизм, race conditions.
12. **websites-servers.md**: Веб-сайты, серверы, процесс загрузки страниц.
13. **frontend-backend.md**: Роли фронтенда и бэкенда в веб-разработке.
14. **http-dns.md**: HTTP-запросы (GET, POST, PUT, DELETE), DNS и кэширование.
15. **databases-basics.md**: Реляционные базы, структура таблиц, ключи.
16. **data-normalization.md**: Нормализация данных, нормальные формы.
17. **nosql-databases.md**: Нереляционные базы (документо-ориентированные, ключ-значение, графовые).
18. **data-caching.md**: Кэширование (браузер, база данных, оперативная память).
19. **deployment-devops.md**: Деплой, автоматизация, DevOps.
20. **ci-cd.md**: Непрерывная интеграция и доставка, настройка задач.
21. **cloud-infrastructure.md**: Облачные сервисы (IaaS, PaaS, SaaS).
22. **anonymity-ip-rotation.md**: Анонимность и автоматическая смена IP-адресов для защиты конфиденциальности и обхода блокировок.
23. **tor-network.md**: Tor сеть для анонимной коммуникации и сокрытия IP-адреса.
24. **proxy-socks5.md**: Настройка прокси-серверов (SOCKS5) для маршрутизации трафика и обеспечения анонимности.
25. **dns-leak.md**: DNS утечки и методы их предотвращения при использовании прокси и VPN.
26. **web-scraping-parsing.md**: Парсинг данных с веб-сайтов, автоматизированное извлечение информации.
27. **osint.md**: OSINT (Open Source Intelligence) — сбор и анализ информации из открытых источников для разведки и пентестинга.
28. **internet-device-scanning.md**: Поиск уязвимых устройств в интернете (IoT, камеры, серверы) с использованием специализированных поисковых систем.
29. **wireless-network-analysis.md**: Анализ Wi-Fi сетей, типов шифрования и выявление сетей с устаревшими методами защиты.
30. **email-enumeration.md**: Поиск email-адресов, связанных с доменами, для OSINT исследований и этичных пентестов.
31. **cve-databases.md**: Базы данных уязвимостей, эксплойтов и патчей (CVE, CWE) для отслеживания известных уязвимостей.
32. **ssl-tls-certificate-analysis.md**: Анализ SSL и TLS сертификатов для выявления потенциально опасных сертификатов и аудита безопасности.
33. **source-code-search.md**: Поиск исходного кода по ключевым словам в публичных репозиториях для обнаружения уязвимостей и секретов.
34. **xss-vulnerability.md**: XSS (Cross-Site Scripting) — уязвимость, позволяющая внедрить вредоносный JavaScript-код на сайт.
35. **sql-injection.md**: SQL Injection — уязвимость, позволяющая внедрить код в SQL-запрос и получить доступ к конфиденциальной информации.
36. **csrf-vulnerability.md**: CSRF (Cross-Site Request Forgery) — уязвимость, позволяющая подменять формы и выполнять действия от имени пользователя.
37. **code-injection.md**: Code Injection — уязвимость, позволяющая выполнять произвольный код на сервере через неправильную обработку ввода.
38. **password-security.md**: Методы защиты паролей: использование соли, хэширование, сложные пароли и двухфакторная аутентификация.
39. **input-validation-sanitization.md**: Валидация и санитизация пользовательского ввода для предотвращения уязвимостей.
40. **cookie-security.md**: Методы защиты кукисов: флаг http_only, правильное хранение токенов аутентификации.
41. **rest-api-security.md**: Безопасность REST API: следование правилам REST, защита от CSRF, безопасное взаимодействие фронтенда с бэкендом.

### Новые инструменты (добавлены в `tools/`)

1. **python.md**: Для написания скриптов, автоматизации, обработки исключений, запуска серверов.
2. **git.md**: Управление версиями кода, работа с репозиториями.
3. **docker.md**: Контейнеризация, автоматизация деплоя, изолированные среды.
4. **tor.md**: Обеспечение анонимного доступа в сеть через распределенную сеть узлов, настройка SOCKS5 прокси.
5. **tornet.md**: Автоматическая смена IP-адресов через Tor сеть с заданным интервалом для парсинга и автоматизации.
6. **shodan.md**: Поиск уязвимых устройств и сервисов в интернете, сканирование подключенных устройств.
7. **wi-fi-map.md**: Анализ беспроводных сетей, типов шифрования и географического расположения точек доступа.
8. **hunter.md**: Поиск email-адресов, связанных с доменами, верификация подлинности адресов.
9. **vulners.md**: База данных уязвимостей, эксплойтов и патчей, отслеживание развития уязвимостей.
10. **certy.md**: База данных SSL и TLS сертификатов, выявление потенциально опасных сертификатов.
11. **full-hunt.md**: Сканирование веб-приложений на уязвимости, автоматические отчёты и рекомендации.
12. **codecrawler.md**: Поиск исходного кода по ключевым словам в публичных репозиториях.
13. **brakeman.md**: Статический анализатор безопасности для Rails приложений, автоматическое обнаружение XSS, SQL Injection, CSRF.
